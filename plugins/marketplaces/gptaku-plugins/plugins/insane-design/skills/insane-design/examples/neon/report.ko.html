Dark carbon + neon electron single spike

Neon은 "Postgres, but electric"의 브랜드 서사를 디자인에 정확히 심어 넣는다. 페이지 기본은 dark #0A0A0A에 가까운 carbon 배경에 neon-green #00E599 (lab(80.7954% -61.4598 23.582)로 정밀 정의된 bio-electric 그린)이 단일 accent로 번진다. #38A57D (focus outline), #00E599 (hover link), #AFB1B6 (muted text) 페어가 핵심.

색상 전략은 "Postgres 로고 돌고래 코랄 → 전자 녹색으로의 재해석". neutral dark 위 단일 neon green spike. .text-with-links a:hover { color: #00E599; border-color: rgba(0,229,153,0.4) } 처럼 link hover에 neon green ring. text-secondary #C9CBCF · text-muted #AFB1B6의 슬레이트 계단.

타이포그래피는 Inter Variable + Geist Mono + esbuild (자체 호스팅 특수 폰트). Vercel 생태계 동조. --font-weight-semibold: 600 + --tw-tracking: .02em (positive tracking!).

레이아웃은 wide + dense. max-width 1440px+. hero는 animated border (.animated-button 관찰 — top/side/bottom mask-image로 electron 애니메이션) + 제품 dashboard live view.

인터랙션이 브랜드 특징. .animated-button은 top/side/bottom mask-image로 #000 → transparent radial 그라디언트 animation을 돌린다 — neon "electron moving along edge" 효과.

  --bg:      #0A0A0A;
  --surface: #141414;
  --fg:      #FAFAFA;
  --fg-muted: #AFB1B6;
  --border:  #27272A;
}
/* 3. Neon green */

  --color-accent:     #00E599;
  --color-accent-soft:rgba(0,229,153,.4);
  --color-focus-ring: #38A57D;
}
a:hover { color: var(--color-accent); }
:focus-visible {
  outline: 2px solid var(--color-focus-ring);
  outline-offset: 2px;
}
절대 하지 말 것: 브랜드 green을 Tailwind emerald #10B981로 대체하지 마라. Neon의 그린은 #00E599 — 더 높은 saturation, lab color space로 정밀 정의된 "전자처럼 빛나는" 값. emerald는 leafy green, neon green은 oscilloscope glow.

Do / Don't

Dark-first electric 규칙.

DO

  • 배경 #0A0A0A dark carbon. pure black 금지.
  • accent green은 #00E599 정밀 neon. emerald 대체 금지.
  • focus outline #38A57D (deeper green) 2px offset 2px.
  • text 계단: #FAFAFA → #C9CBCF → #AFB1B6.
  • tracking .02em positive (wide) 히어로.
  • animated-button electron 애니메이션 복제.

DON'T

  • Tailwind emerald #10B981 쓰지 말 것.
  • monochrome tight tracking -0.02em 쓰지 말 것 (히어로는 wide).
  • focus ring을 blue로 쓰지 말 것.
  • 카드 shadow 강하게 금지.
  • light 테마 기본 금지.
