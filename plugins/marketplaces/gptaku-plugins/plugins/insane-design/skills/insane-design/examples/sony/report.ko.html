Design System Report

Sony — SST W20 · Dark 모노크롬

https://www.sony.com 이 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. 순흑(#000000) 배경의 dark 테마가 기본 — 파란색은 PlayStation 서브브랜드 전용이다.

Quick Start 5분 안에 Sony처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight 400 body { font-family: "SST W20 Roman", "SST W55 Regular", Arial, sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 (dark) :root { --bg: #000000; /* pure black */ --fg: #FFFFFF; /* pure white */ } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 컬러 :root { /* 모노크롬 — 파랑 금지 */ --brand: #000000; /* CTA: bg #FFFFFF, color #000 */ }
절대 하지 말아야 할 것 하나: Sony UI에 파란색(#003087)을 넣는 것. 현재 Sony.com은 블랙 기반 dark 테마가 주류이며, 파랑은 PlayStation 등 서브브랜드에만 존재한다. 기본 Sony.com UI는 순수한 흑백 모노크롬이다.

03 · Typography

타이포그래피

SST 폰트 패밀리 — weight별 별도 파일. font-weight: 700이 아닌 font-family: "SST W20 Bold"로 굵기 제어.

Font stack
Display font SST W20 Bold (Sony 독점 라이선스)
Body font SST W20 Roman (Sony 독점)
Light font SST W20 Light (Sony 독점)
Weight normal / bold 400 / 700
라이선스 주의: SST (Sharp Sans Text) 시리즈는 Sony 독점 폰트. 대체재로 Sharp Grotesk 또는 Helvetica Neue 사용. 또한 SST는 weight별 별도 파일로 관리 — font-weight: 700이 아니라 font-family: "SST W20 Bold"로 굵기를 제어한다.

12 · Do / Don't

Do / Don't

Sony 디자인 시스템의 핵심 규칙.

✓ DO

  • 기본 배경은 #000000(pure black)으로 설정한다
  • dark 배경에서 primary CTA는 background: #FFFFFF, color: #000000
  • heading은 font-family: "SST W20 Bold"로 별도 지정한다
  • 제품 이미지 배경은 #000000 또는 #1A1A1A 유지
  • 그림자는 불투명도를 30% 이상으로 설정한다 (dark 배경 가시성)

✗ DON'T

  • 파란색을 기본 Sony.com UI에 넣지 않는다 — PlayStation 서브브랜드 전용
  • font-weight: 700만으로 굵기를 조절하지 않는다 — SST Bold 폰트 파일 사용
  • 배경을 밝은 회색(#F5F5F5)으로 두지 않는다 — Sony는 dark 테마 기본
  • 불필요한 색상 악센트를 UI에 추가하지 않는다 — 모노크롬 시스템
  • border-radius에 큰 값(16px 이상)을 버튼에 쓰지 않는다
잘못된 구현
올바른 구현

color: #FFFFFF or #000000

font-weight: 700
font-family: "SST W20 Bold"
SST는 weight별 별도 파일. font-weight가 아닌 font-family로 굵기를 제어한다.
background: #F5F5F5
background: #000000
Sony 기본 테마는 dark. 밝은 배경은 Sony.com의 정체성을 훼손한다.
btn { background: #000000 } (dark bg에서)
btn { background: #FFFFFF; color: #000000 }
dark 배경에서 primary CTA는 흰 배경. 검정 버튼은 배경과 구분이 안 됨.
