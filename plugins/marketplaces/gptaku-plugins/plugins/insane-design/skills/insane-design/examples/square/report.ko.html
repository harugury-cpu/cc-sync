Design System Report

Square — --color-* / Svelte DS

https://squareup.com 이 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. SvelteKit으로 구축된 Square 웹사이트의 Svelte scoped CSS와 `--color-*` 토큰 체계를 분석했다. 모든 값은 CSS 파싱 결과이며, AI 추론은 섞이지 않았다.

Quick Start 5분 안에 Square처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight 400 body { font-family: 'Square Sans Text VF', 'Square Sans Text', Helvetica, Arial, sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #FFFFFF; --fg: #1A1A1A; } body { background: var(--bg); color: var(--fg); }
3 · Square 블루 :root { --color-theme: #006AFF; } /* primary CTA 전용 */
절대 하지 말아야 할 것 하나: Cash Sans를 Square의 메인 UI 폰트로 사용하는 것. Cash Sans는 Square의 Cash App 제품라인 전용 커스텀 폰트다. Square 웹사이트의 메인 UI 폰트는 Square Sans Text VF이며, 두 폰트를 혼용하면 완전히 다른 두 제품의 아이덴티티가 충돌한다.

04 · Colors

컬러

CSS 커스텀 프로퍼티 파싱 + Svelte scoped selector_role 확인. #006AFF는 primary CTA와 alt 버튼 border 두 곳에서 모두 확인.

Brand Blue — --color-theme · primary CTA 배경

--color-theme-focus / hover #0055CC
블루 tint 배경 #F0F6FF
Neutrals — 라이트 테마
메인 텍스트 #1A1A1A
보더 #D9D9D9
accent 배경 (light) #F7F6F5
기본 배경 #FFFFFF
Dark Mode accent
--color-background-accent-color-mode-dark #1F1F1F
--color-theme 패턴: Square의 primary 버튼은 background-color: var(--color-theme, #006aff)로 설정된다. CSS 변수를 사용하면서 fallback으로 hex값을 명시하는 패턴. 이 패턴을 그대로 복사해야 Square의 테마 시스템과 호환된다.

13 · Do / Don't

Do / Don't

Square 디자인 시스템을 올바르게 구현하기 위한 핵심 원칙.

DO

  • var(--color-theme, #006AFF) 패턴 — 변수 + fallback 항상 함께
  • 버튼 ::before overlay 패턴으로 hover 표현
  • Square Sans Text VF로 UI 전체 — 압도적 주폰트
  • Square Sans Display로 히어로 헤드라인
  • 버튼 radius 4px — 비즈니스 신뢰감
  • CSS 변수 --color-border: #D9D9D9 패턴 유지

DON'T

  • Cash Sans를 메인 UI 폰트로 사용 — Cash App 전용
  • pill shape 버튼 (border-radius: 999px) — 비즈니스 느낌 훼손
  • 다크 배경 기본 사용 — 라이트가 Square 주 테마
  • --color-theme 없이 hex만 하드코딩 — 변수 패턴 유지 필요
  • Svelte 해시 클래스(.svelte-*) 직접 작성 — 컴파일 결과물
오해 실제 설명
Cash Sans = Square 메인 폰트 Square Sans Text VF Cash Sans는 Cash App 전용. Square 웹사이트 메인은 Square Sans Text VF (64회).
결제사 = 검정/흰 무채색 #006AFF 블루 CTA 라이트 테마 + #006AFF 블루 CTA. 흰 배경에 블루 버튼의 신뢰감.
버튼이 둥글 것 (핀테크 = pill) 4px 작은 radius Square는 결제 처리 회사. 샤프하고 비즈니스적인 4px radius. Wise pill과 완전히 다름.
Svelte 해시 = 의미 없는 CSS 브랜드 컬러 직접 확인 .primary.svelte-wxk8im에서 #006AFF 직접 확인. Svelte scoped CSS도 파싱 대상.
