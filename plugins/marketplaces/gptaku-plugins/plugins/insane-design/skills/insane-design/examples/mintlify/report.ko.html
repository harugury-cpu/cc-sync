차분한 warm neutral + mint green 단점

Mintlify는 개발자 문서를 위한 플랫폼이다. 디자인은 이를 그대로 반영해 — 차분한 light 기본 + 녹색 브랜드 + Geist Mono 코드. 페이지는 #FFFFFF 또는 warm #F7F7F7 배경에 #231F20 (warm black) 텍스트가 올려진다. 다크 모드 #151616 배경도 존재. 브랜드 green #0C8C5E (light) / #18E299 (dark)가 단일 anchor.

색상 전략은 "문서가 읽혀야 한다". neutral 위주 팔레트 + 단일 mint green accent + warm neutral gradient. footer CTA 영역에서 linear-gradient(#F7F7F7 0%, rgba(247,247,247,0.95) 4.7%, ... 0%)처럼 다중 stop gradient fade로 섹션 경계를 부드럽게 — Mintlify 시그니처.

타이포그래피는 Inter + Geist Mono — Vercel/Next.js 생태계 default. weight 400/500/600/700/800/900까지 사용. 히어로는 800 또는 900의 full weight로 임팩트.

레이아웃은 좁은 prose 중앙. docs 플랫폼답게 prose 720px 기본. Tailwind v4 (--tw-* variable system 감지).

인터랙션은 거의 없다 — 문서 사이트는 정적인 게 미덕. hover color 변화 + subtle transition만.

Mintlify — --color-brand / Tailwind v4

Docs-first 디자인 시스템. warm neutral + mint green single-accent + multi-stop gradient fade.

/* 1. 폰트 */
body {
  font-family: "inter",
    "Inter Fallback",
    -apple-system,
    BlinkMacSystemFont,
    "Segoe UI",sans-serif;
  font-weight: 400;
}
/* 2. Light bg + text */
:root {
  --bg:         #FFFFFF;
  --bg-warm:    #F7F7F7;
  --fg:         #231F20;
  --fg-muted:   #4A5565;
  --border:     #E5E7EB;
}
.dark {
  --bg: #151616;
  --fg: #F7F7F7;
}
/* 3. Brand mint */
:root {
  --color-brand: #0C8C5E;
  --color-brand-light: #0C8C5E;
}
.dark {
  --color-brand: #18E299;
}
절대 하지 말 것: mint green #0C8C5E을 넓은 배경 면적에 쓰지 마라. 문서 사이트 브랜드는 neutral 위에 한 점의 초록이 포인트일 때만 성립. 녹색 배경은 Shopify·Duolingo지, Mintlify가 아니다.

Do / Don't

Docs-first 규칙.

DO

  • light 기본 #FFFFFF · 텍스트 #231F20 warm black.
  • 브랜드 mint #0C8C5E은 link/CTA/code-token-color에만.
  • Inter weight 400/600/900 3단 계단.
  • 카드 radius 12px, 버튼 6px.
  • footer-cta-gradient multi-stop fade 사용.
  • 다크는 --color-brand: #18E299 bright mint.

DON'T

  • mint를 넓은 배경에 쓰지 말 것 — Duolingo 됨.
  • weight 800–900을 본문에 쓰지 말 것 — display 전용.
  • monochrome 엄격히 가지 말 것 — #F7F7F7 + #F1EFED 레이어.
  • 단일 stop gradient 금지 — multi-stop fade.
  • prose max-width 1024px 이상 금지.
