Kakao — KakaoBig · 카카오 시그니처 노랑

www.kakaocorp.com 의 CSS. KakaoBig/KakaoSmall 전용 폰트 이분법. #FEE500 노랑은 배경 전용 — 텍스트는 반드시 #000000.

Quick Start5분 안에 Kakao처럼 만들기
1 · 폰트 이분법/* 대제목 */ body.headline { font-family: KakaoBig, sans-serif; } /* 본문 */ body { font-family: KakaoSmall, sans-serif; }
2 · 배경 + 텍스트:root { --bg: #FFFFFF; --fg: #333333; } body { background:var(--bg); color:var(--fg); }
3 · 브랜드 노랑:root { --brand: #FEE500; /* 텍스트 on-yellow */ --on-brand: #000000; } /* 절대 yellow text X */
절대 하지 말아야 할 것 하나: #FEE500 카카오 노랑을 텍스트 색으로 사용. #FEE500은 배경 전용 — 실제 CSS .btn_play의 color는 #000000이며, 노랑 위에는 항상 검정 텍스트가 들어간다.

08 · Do / Don't

자주 틀리는 것들

Kakao 웹 구현 시 가장 많이 실수하는 패턴.

✓ DO

  • 카카오 노랑 #FEE500 위에는 항상 검정 #000000 텍스트
  • 대제목은 KakaoBig, 본문은 KakaoSmall 구분
  • btn_play 패딩 비대칭 유지 — top 9px, bottom 11px
  • 아이콘 stroke 색 #888888

✗ DON'T

  • #FEE500 노랑을 텍스트 색으로 사용 — 배경 전용
  • KakaoBig과 KakaoSmall을 하나의 font-family로 통합
  • border-radius 8px 이상 — Kakao CSS 기준 6px가 최대
  • white 텍스트를 노랑 버튼 위에 올리기 — 명도 대비 부족
