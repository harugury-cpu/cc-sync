기능 밀도 최우선 — 실용적 이커머스 UI

쿠팡은 정보 전달이 미감보다 우선이다. 흰 배경 위에 브랜드 블루와 프로모 레드가 능동적 요소에 집중 투입된다.

쿠팡은 한국 최대 이커머스 플랫폼답게 기능 밀도 최우선의 실용적 UI를 구사한다. 흰 배경 #FFFFFF 위에 짙은 텍스트 #212B36이 촘촘하게 배치되고, 쿠팡 시그니처 블루 #346AFF가 CTA·링크·배지 등 모든 능동적 요소에 집중 투입된다.

색상 전략은 단순하다: 배경은 #FFFFFF 순백, 텍스트는 #212B36(다크 잉크), 브랜드 블루 #346AFF가 CTAㆍ탭ㆍ강조에 독점 사용된다. 빨강 #CB1400은 프로모션 가격과 긴급 배지에 한정되고, 중간 회색이 구분선·보조 텍스트를 담당한다.

한국어 UI 최적화를 위해 Apple SD Gothic Neo를 1순위 폰트로, 다국어 폴백으로 Noto Sans KR·Roboto·Helvetica Neue를 순서대로 연결한다. 웨이트는 주로 400(body)·700(제목·강조)로 양극화.

여백은 소형 단위 중심(4px·8px·12px·16px)이며 max-width: 1600px의 넓은 컨테이너 안에서 촘촘한 그리드로 상품을 가득 채운다. 그림자(box-shadow)는 CSS에 거의 없고, 카드 구분은 테두리 1px로 한다.

기능 밀도 최우선 #346AFF 브랜드 블루 Apple SD Gothic Neo 1px border (shadow 없음) 0.15s transition

레이아웃 패턴 — 1600px 최대, 촘촘한 상품 그리드

Grid System

  • Content max-width: 1600px
  • Grid type: CSS Grid + Flexbox
  • Columns: 최대 6열 (상품 그리드)
  • Gutter: 16px

Card Patterns

  • Background: #FFFFFF
  • Border: 1px solid #DDDDDD
  • Radius: 4px–8px

  • Shadow: 없음 (border 전용)

Navigation

  • Type: 수평 메가메뉴 + 카테고리 사이드바

  • Height: 56px

반복 작업 가이드
  • • 색상 변경 시: #346AFF 브랜드 블루 기준. 프로모는 #CB1400. 두 색만 사용.
  • • 폰트 변경 시: weight 400 기본. 700은 가격·CTA에만.
  • • 여백 조정 시: 4의 배수(4/8/12/16/24px) 단위로만.
  • • 새 컴포넌트: 1px solid #DDDDDD border + 4px radius 패턴을 따를 것.
  • • 반응형: 768px·1024px·1600px 기준.
15

Do / Don't

✅ DO

  • • CTA 버튼에 #346AFF 정확히 사용
  • • 한국어 UI에 Apple SD Gothic Neo 1순위
  • • 할인가·배지에 #CB1400 레드
  • • 카드 구분: 1px solid #DDDDDD 테두리만
  • • 버튼 radius: 4px
  • • font-weight: 400(body)·700(강조) 두 단계만

❌ DON'T

  • • #0066CC 등 임의 파랑 사용 금지
  • • 배경에 임의 회색 금지 — #F5F5F5만
  • • box-shadow 남발 금지 — border 전용
  • • 버튼 radius 8px 이상 금지
  • • 폰트 크기 10px 이하 금지
  • • 다크 모드 구현 시도 금지
