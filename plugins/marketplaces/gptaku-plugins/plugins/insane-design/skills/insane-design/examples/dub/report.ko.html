순백 캔버스 위 검정 CTA + violet accent

Dub의 마케팅 홈은 Refined Minimalism 방향성을 유지한다. 브랜드 컬러 #855AFC는 CTA·링크·focus ring 등 의미가 필요한 지점에만 등장하며, 넓은 면적은 #FFFFFF 계열이 담당한다.

링크 어트리뷰션 분석 플랫폼 — 순백 배경 + 얇은 그리드 + 검정 CTA + 보라 한 점.

타이포그래피는 Geist를 기본 축으로 한다. body는 400 weight + 16px, H1은 48~64px의 8단 스케일. 큰 사이즈에 음수 tracking(-0.02em ~ -0.03em) 적용으로 optical compensation.

레이아웃은 1200-1440px max-width + 8px baseline 체계. 섹션 리듬 64-120px vertical. 모션은 150-200ms + ease-out 기본.

Dub — --bg-*, --border-*, --content-*

링크 어트리뷰션 분석 플랫폼 — 순백 배경 + 얇은 그리드 + 검정 CTA + 보라 한 점.

/* 1. 폰트 + weight */
body {
  font-family: "Geist", -apple-system, sans-serif;
  font-weight: 400;
  font-size: 16px;
}
/* 2. 배경 + 텍스트 */
:root {
  --bg: #FFFFFF;
  --fg: #171717;
  --border: #E5E5E5;
}
body {
  background: var(--bg);
  color: var(--fg);
}
/* 3. 브랜드 악센트 */
:root {
  --brand: #855AFC;
}
.cta {
  background: var(--brand);
  color: #FFFFFF;
}
절대 하지 말 것: 브랜드 색 #855AFC를 본문 텍스트 또는 긴 문단 배경으로 쓰지 말 것. Dub의 정체성은 accent를 "한 점"에만 올리는 절제다.

Do / Don't

실제 CSS에서 검증된 규칙만 수록.

DO

  • ✅ 브랜드 색 #855AFC는 CTA·링크·focus ring에만 사용
  • ✅ 배경은 #FFFFFF 또는 #FAFAFA (panel)
  • ✅ 본문 텍스트 #171717, muted #737373
  • ✅ H1~H3에 음수 letter-spacing 적용 (-0.01em ~ -0.03em)
  • ✅ Radius 4/8/12/16/24/9999px 6단 체계 유지

DON'T

  • ❌ 배경을 #EEEEEE 중간 회색으로 두지 말 것 — 대신 #FAFAFA 사용
  • ❌ 텍스트를 #000000/black으로 두지 말 것 — 대신 #171717
  • ❌ body에 font-weight: 300 사용 금지 — Dub은 400이 기본
  • ❌ 브랜드 #855AFC를 긴 문단 배경에 쓰지 말 것
  • ❌ Radius 20px, 30px 같은 외부 값 금지
  • ❌ 단층 shadow 0 4px 12px rgba(0,0,0,0.1) 금지 — 2-layer stack
  • ❌ transition duration 400ms+ 금지 — 150-200ms 유지
