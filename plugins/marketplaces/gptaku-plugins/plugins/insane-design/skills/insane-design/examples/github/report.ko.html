Primer semantic token + 접근성 테마 분기

GitHub의 랜딩/홈은 Primer Design System의 산물이다. 20개가 넘는 테마 CSS 파일(dark, dark_colorblind, dark_dimmed, dark_high_contrast, dark_tritanopia, light 동일 계열)이 독립 팔레트로 분리된 것 자체가 시그니처 — GitHub은 접근성(WCAG)을 디자인 시스템의 1급 시민으로 대우한다.

색상 전략은 "의미론적 semantic token이 전부". --fgColor-*, --bgColor-*, --borderColor-*, --button-* 같은 role 이름 alias만 UI 코드에 쓰이고, raw hex는 --base-color-* 1-tier에만 존재한다. 브랜드 green #2DA44E + accent blue #1F6FEB가 핵심. 파란색은 link/focus/accent, 초록은 primary CTA + success.

타이포그래피는 Mona Sans (GitHub 자체 variable font, OFL) + Hubot Sans + system-ui. 코드는 SF Mono / Consolas. weight 400/500/600 주류에 hero 700.

레이아웃은 좁은 content 중앙 + 양옆 여백 — max-width 1280px. hero는 dark gradient + cyan accent glow. 카드는 radius 6–12px 소박. shadow는 정교한 dual-layer.

인터랙션은 절제된 즉시성. button hover는 bg 변화만 (#2E9A40 → #29903B). 긴 transition 없음.

GitHub — --fgColor-* / --bgColor-*

URL 하나로 추출한 Primer 디자인 시스템. semantic alias 수백 개 + 8개 접근성 테마 분기.

/* 1. 폰트 + weight */
body {
  font-family: "Mona Sans",
    "Hubot Sans",
    -apple-system,BlinkMacSystemFont,
    "Segoe UI","Noto Sans",sans-serif;
  font-weight: 400;
}
/* 2. Dark default */
:root {
  --bgColor-default:  #0D1117;
  --bgColor-muted:    #010409;
  --bgColor-subtle:   #161B22;
  --fgColor-default:  #E6EDF3;
  --fgColor-muted:    #8B949E;
  --borderColor-default: #30363D;
}
/* 3. Primary green + accent */
:root {
  --button-primary-bgColor-rest: #2DA44E;
  --button-primary-bgColor-hover:#29903B;
  --fgColor-accent:              #4493F8;
  --bgColor-accent-emphasis:     #1F6FEB;
}
절대 하지 말 것: raw hex #0D1117, #2DA44E를 컴포넌트 CSS에 직접 쓰지 마라. GitHub은 무조건 semantic alias (--bgColor-default, --button-primary-bgColor-rest)로만 참조한다. raw hex 사용은 테마 분기를 불가능하게 만든다.

Do / Don't

Primer 검증 규칙.

DO

  • 컴포넌트 CSS는 --bgColor-* / --fgColor-* / --button-* alias로만.
  • primary CTA는 #2DA44E green, hover #29903B.
  • 본문은 Mona Sans, body 14px (UI) 또는 16px (marketing).
  • 카드 radius 6px (기본), 12px (큰 것).
  • border는 #30363D (dark) / #D1D9E0 (light).
  • 접근성: data-theme="dark_colorblind" 처럼 스왑.

DON'T

  • raw hex #0D1117를 컴포넌트 CSS에 직접 쓰지 말 것.
  • primary bg에 파랑 #1F6FEB 쓰지 말 것 — primary는 green.
  • radius 8px 쓰지 말 것 — Primer는 4/6/12.
  • weight 800/900 쓰지 말 것 — 최대 700.
  • 카드 shadow 강하게 쓰지 말 것.
