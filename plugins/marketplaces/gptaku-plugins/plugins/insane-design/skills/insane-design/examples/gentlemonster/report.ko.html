Gentle Monster — Custom Fonts

gentlemonster.com 실제 CSS에서 뽑은 디자인 토큰. 흑백 2색 시스템과 weight 350 전용 Light 폰트가 럭셔리 감성을 만든다.

Quick Start 5분 안에 Gentle Monster처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight 350 body { font-family: Arial, sans-serif; font-weight: 350; /* GM Light 계열 대체: weight 300 */ }
2 · 흰 배경 + 검정 텍스트 :root { --bg: #FFFFFF; --fg: #111111; } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 컬러 :root { --brand: #111111; } /* 흑백 단색 시스템. accent 색 없음. */
절대 하지 말아야 할 것 하나: font-weight: 400을 기본으로 두는 것. Gentle Monster UI의 정체성은 weight 350의 Light 계열 폰트다. 400으로 쓰면 럭셔리 감성이 무너진다.

10 · Do / Don't

자주 틀리는 것들

Gentle Monster 디자인을 구현할 때 흔히 실수하는 패턴.

✓ DO

  • font-weight: 350을 정확히 써라 — GM 폰트 전용 비표준 weight다.
  • 배경은 순백 #FFFFFF으로 써라 — GM은 라이트 베이스다.
  • 텍스트와 CTA 버튼 배경은 모두 #111111로 통일하라.
  • 흑백 2색 시스템을 유지하라 — 브랜드 accent 색이 없어도 강하다.
  • 버튼에 border-radius: 0을 써라 — GM은 각진 디자인이다.

✗ DON'T

  • font-weight: 400을 기본으로 두지 마라 — GM의 기본은 350이다.
  • 밝은 컬러(파랑, 빨강)를 accent로 쓰지 마라 — GM은 흑백 시스템이다.
  • 모서리를 과도하게 둥글게 만들지 마라 — 일반 버튼은 radius 0이다.
  • 로고/제품 이미지의 hex를 UI 색으로 쓰지 마라 — UI에 적용되지 않는다.
  • #858585를 일반 텍스트에 쓰지 마라 — disabled 전용이다.
