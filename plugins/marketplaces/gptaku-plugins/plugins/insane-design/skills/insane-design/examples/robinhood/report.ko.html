Design System Report

Robinhood — Capsule DS

https://robinhood.com 이 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. 모든 값은 CSS 주파수 파싱 결과이고, AI 추론이나 추측은 섞이지 않았다.

Quick Start 5분 안에 Robinhood처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight 400 body { font-family: 'Capsule Sans Text', Phonic, Helvetica, sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #110E08; /* 다크 브라운 */ --fg: #FFFFFF; } body { background: var(--bg); color: var(--fg); }
3 · 네온 옐로우-그린 :root { --brand: #CCFF00; } /* CTA 버튼 */ .btn { background: #CCFF00; color: #110E08; /* 다크 텍스트 */ }
절대 하지 말아야 할 것 하나: #CCFF00을 흰 배경에서 사용하는 것. 다크 브라운 #110E08 배경 위에서만 네온이 제대로 발광한다. 흰 배경에서는 전혀 다른 브랜드가 된다.

13 · Do / Don't

Do / Don't

Robinhood 디자인 시스템의 핵심 규칙과 자주 하는 오해.

✓ DO

  • #CCFF00 네온을 다크 배경 위 CTA에만 사용
  • CTA 텍스트는 #110E08 — 네온 위 다크 텍스트
  • 다크 브라운 #110E08이 필수 배경 컨텍스트
  • Nib Pro Display weight 400으로 우아한 히어로 타이틀
  • 주가 상승에 #CCFF00 적용 — 긍정 연상 강화

✗ DON'T

  • 흰 배경에 #CCFF00 — 다크 배경 없으면 의미 없음
  • CTA 텍스트 흰색 — 네온 배경엔 다크 텍스트
  • 부드러운 그린으로 대체 — #CCFF00은 극단적 네온
  • 굵은 sans-serif 히어로 — weight 400 serif가 Robinhood 언어
  • 작은 radius 버튼 — pill이 정체성
오해 (자주 하는 실수) 실제 이유
Robinhood = 초록 #CCFF00 네온 #CCFF00은 옐로우에 가까운 극단적 네온. 일반 그린이 아님.
흰 배경이 기본 #110E08 다크 마케팅 사이트는 다크 브라운이 primary. 네온의 컨텍스트.
CSS 토큰 기반 DS Emotion 해시 클래스 CSS-in-JS Emotion 사용. 커스텀 프로퍼티(--*)를 사용하지 않음.
