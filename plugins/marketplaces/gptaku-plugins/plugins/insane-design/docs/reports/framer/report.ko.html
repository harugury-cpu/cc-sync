00

near-black 위 cyan electron

Framer의 마케팅 사이트는 "캔버스 위에서 살아 움직이는 노코드 디자인 툴"의 정체성을 시각적으로 증명한다. 전체 페이지는 거의 순흑에 가까운 #080808, #090909, #141414의 3단 dark 레이어로 깔리고, 그 위에 시그니처처럼 반복되는 cyan #0099FF (#09F)가 포인트 악센트로 등장한다.

색상 전략은 "모션이 컬러다"다. 정적인 팔레트로는 dark + cyan 단색처럼 보이지만, 히어로 전환부에서 gradient (#05f → #60f → #90f)와 accent (#FFEEAA butter, #FF7EB0 pink) 점광이 interaction 시에만 번진다. #ffffff1a, #fffc, #fff9, #fff6 같은 white-alpha 토큰을 층층이 겹쳐 glass/surface layering을 만든다.

타이포그래피는 Inter + Inter Display + Inter Tight Inter family 3종을 축으로 삼는다. 히어로는 Inter Display 큰 타이틀, 본문은 Inter, 강조는 Inter Variable. 코드는 JetBrains Mono / Geist Mono.

레이아웃은 canvas 그 자체를 은유한다. scroll-snap 류 scene 연결, full-bleed 이미지·비디오 배경, 카드 radius 16–24px. shadow 대신 bg layering으로 elevation.

인터랙션이 브랜드다. hover에 반드시 motion이 따라온다 — transform scale, opacity fade, gradient shimmer. transition .2s–.4s ease-out. focus outline은 #0099FF cyan 2px.

Framer — --framer-*

URL 하나로 추출한 실제 CSS 토큰 기반 디자인 시스템 리포트. UUID 기반 토큰 + semantic alias 2-tier.

/* 1. 폰트 + weight */
body {
  font-family: "Inter",
    "Inter Variable",
    "SF Pro Display",
    -apple-system,sans-serif;
  font-weight: 400;
}
/* 2. Dark 레이어 */
:root {
  --framer-bg:      #080808;
  --framer-surface: #141414;
  --framer-fg:      #FFFFFF;
  --framer-fg-muted:#fff9;
}
body {
  background: var(--framer-bg);
  color: var(--framer-fg);
}
/* 3. Cyan accent */
:root {
  --framer-accent: #0099FF;
  --framer-border: #ffffff1a;
}
절대 하지 말 것: 모든 dark 배경을 #000000로 두지 마라. Framer는 #080808 / #090909 / #141414 / #171717 4–5단 미세 레이어로 panel 겹침을 만든다. 단일 black은 OLED TV, Framer가 아니다.

  • 폰트 Inter Display (hero) + Inter (body). weight 400/500/600.
  • 여백 4px 배수 16/24/32/48/64/96.

  • 다크 유일 — light는 보조.

  • 모션 hover에 반드시 subtle transform.

Do / Don't

실제 CSS 검증 규칙.

DO

  • 배경은 #080808 또는 #090909.
  • 다단 레이어 (#080808 → #141414 → #171717) 겹치기.
  • cyan accent #0099FF는 link/focus/CTA에만.
  • border는 #ffffff1a (white 10% α) 소프트.
  • 본문 weight 400, 타이틀 600.
  • 카드 radius 24px, 버튼 12px.
  • hover에 .2s ease-out transform scale 추가.

DON'T

  • pure black #000000 배경 금지.
  • cyan을 넓은 면적에 쓰지 말 것.
  • weight 700 금지 — 600 상한.
  • radius 4px/6px 금지.
  • light-theme shadow 0 4px 12px rgba(0,0,0,0.1) 금지.
  • hover static 금지 — 반드시 motion.
  • light-first 테마 금지.
