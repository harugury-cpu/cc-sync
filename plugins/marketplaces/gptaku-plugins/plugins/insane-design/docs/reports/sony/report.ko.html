Design System Report

Sony — SST W20 · mixed monochrome

https://www.sony.com/en/의 live CSS와 공식 브랜드 CSS를 함께 대조해 Sony의 실제 frame palette, SST 타이포그래피, 레거시 red utility만 남긴 리포트. 홈 런타임의 핵심은 #000000 프레임과 #FFFFFF 컨텐츠, 그리고 #656565 body text다.

Quick Start 5분 안에 Sony처럼 만들기 — 3가지만 하면 80%
1 · SST 본문 + Bold 분리 body { font-family: "SST W20 Roman", "SST W55 Regular", "Yu Gothic Medium", YuGothic, Arial, sans-serif; font-size: .875rem; color: #656565; }
2 · mixed frame palette :root { --frame: #000000; --surface: #FFFFFF; --surface-subtle: #EFEFEF; --border: #BFBFBF; --accent-red: #CC0000; }
3 · 1190px shell .page-shell { max-width: 1190px; margin: 0 auto; padding-inline: 20px; } section { padding: 64px 0 80px; }
절대 하지 말아야 할 것 하나: Sony를 full-dark 또는 neon-red 사이트로 만들지 마라. 실제 홈은 검은 frame + 흰 content 구조이며, 2026-04-23 기준 Sony-owned CSS에서 #FF0000는 확인되지 않았다. red는 브랜드 CSS의 #CC0000 utility 수준에서만 검증됐다.

02 · Tech Stack

테크 스택

Sony의 공개 home bundle은 토큰 시스템보다 글로벌 템플릿과 페이지 모듈의 조합에 가깝다.

FRAMEWORK
server-rendered corporate portal + 정적 CSS 경로 /template/2023, /template/2024, /top/2021
DESIGN SYSTEM
공개 토큰형 DS 없음. tmpl-* 글로벌 프레임 + hero-*, panel-*, news-* 모듈 조합
CSS ARCHITECTURE
semantic class naming + fixed-width shell + vendor CSS 병행
DEFAULT THEME
mixed — black header/hero/footer + white content + gray sitemap
FONT LOADING
SST W20 Roman, SST W20 Bold, SST W20 Light, SST W55 Regular/Bold
CANONICAL ANCHOR
#000000 — header, hero, scroll guide, flyout nav의 공통 frame
LAYOUT RULE
1190px fixed inner width + 20px outer gutter
MOTION
.3s / .38s / .48s transition과 scroll-guide bounce만 사용
03 · Typography

타이포그래피

Sony 홈의 핵심은 14px 본문과 SST W20 Bold family 분리다. 굵기를 weight만으로 흉내 내면 질감이 달라진다.

Font stack
Body / UI "SST W20 Roman", "SST W55 Regular", "Yu Gothic Medium", YuGothic, sans-serif
Strong / Bold "SST W20 Bold", "SST W55 Bold", Arial, sans-serif
Light variant "SST W20 Light", sans-serif (brand CSS verification)
Locale helper "SST Japanese"
Weight model 400 / 700, 하지만 강조는 family 교체와 함께 사용
확인된 패턴: .b { font-family: "SST W20 Bold"; font-weight: 700; }. Sony는 font-weight: 700만 쓰지 않고, 실제 굵은 family를 별도로 잡는다.

12 · Do / Don't

Do / Don't

Sony를 Sony답게 보이게 만드는 건 black frame, compact SST, 얇은 border다.

✓ DO

  • 70px black masthead와 #363636 divider를 유지한다
  • 본문은 14px SST W20 Roman + #656565로 시작한다
  • hero는 black shell + blur background + dark overlay 구조를 쓴다
  • 1190px fixed shell과 20px gutter를 지킨다
  • radius는 4px 이하로 제한한다

✗ DON'T

  • Sony를 full-dark 한 장짜리 surface로 단순화하지 않는다
  • font-weight: 700만으로 SST Bold를 흉내 내지 않는다
  • taxonomy color를 primary brand tone처럼 넓게 쓰지 않는다
  • soft radius, floating shadow, glass effect를 추가하지 않는다
  • #FF0000를 검증 없이 primary brand token으로 채택하지 않는다
