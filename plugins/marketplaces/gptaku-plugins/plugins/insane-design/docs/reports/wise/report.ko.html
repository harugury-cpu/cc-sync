Design System Report

Wise — --color-*

https://wise.com 이 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. 모든 값은 CSS 커스텀 프로퍼티를 파싱한 결과이고, AI 추론이나 추측은 섞이지 않았다.

Quick Start 5분 안에 Wise처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight 400 body { font-family: 'Wise Sans', Inter, Helvetica, Arial, sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #FFFFFF; --fg: #0E0F0C; } body { background: var(--bg); color: var(--fg); }
3 · 라임 그린 accent :root { --color-interactive-accent: #9FE870; } /* 라이트 CTA 버튼 */ button { background: #163300; }
절대 하지 말아야 할 것 하나: VideoJS를 Wise의 브랜드 폰트로 착각하는 것. VideoJS(49회)는 미디어 플레이어 아이콘 폰트. 실제 브랜드는 Wise Sans, 본문은 Inter.

13 · Do / Don't

Do / Don't

Wise 디자인 시스템의 핵심 규칙과 자주 하는 오해.

✓ DO

  • #9FE870 라임 그린 — Wise 브랜드 시그니처
  • 버튼 pill shape (border-radius: 40px) — 브랜드 정체성
  • 라이트 CTA: #163300 배경 + #9FE870 텍스트
  • 환율·수수료 수치를 전면 노출 — 투명성이 가치
  • Wise Sans 또는 Inter로 헤드라인

✗ DON'T

  • VideoJS를 UI 폰트로 사용 — 미디어 아이콘 폰트
  • 버튼에 작은 radius — pill이 Wise 언어
  • 수수료를 작게 표시 — 투명성 원칙 위반
  • 일반 그린으로 #9FE870 대체 — 절제된 라임이 핵심
  • 파랑을 브랜드로 — 시안(#00A2DD)은 링크 accent
오해 (자주 하는 실수) 실제 이유
VideoJS가 주폰트 Wise Sans + Inter VideoJS(49회)는 미디어 플레이어 아이콘 폰트. 브랜드 폰트와 무관.
파랑이 브랜드 #9FE870 라임 시안 #00A2DD는 링크/accent 전용. 브랜드 시그니처는 라임 그린.
일반 버튼 radius 40px pill Wise 버튼의 pill shape은 핀테크 브랜드 중 가장 독특한 식별자.
