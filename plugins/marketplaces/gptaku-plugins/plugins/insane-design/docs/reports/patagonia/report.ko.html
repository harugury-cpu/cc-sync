live font check: Ridgeway Sans / Copernicus
Patagonia live CSS

흑백이 먼저고, 자연색은 늦게 나온다. 현재 Patagonia 홈은 `Avenir`가 아니라 `Ridgeway Sans` + `Copernicus`, 기본 UI 앵커는 `#000` / `#FFF`다.

`https://www.patagonia.com/home/` 에서 직접 가져온 Demandware CSS 8개와 인라인 스타일 3개를 다시 읽어, 실제 사용 중인 폰트·색상·스페이싱·컴포넌트 규칙만 정리했다. 기존 예시의 `Avenir Next W02 Light` 가설은 live CSS 기준으로 성립하지 않는다.

확인 결과 `Avenir Next W02 Light`는 2026-04-23 live HTML/CSS에 없다. 예시 폴더의 기존 `_inline.css`는 site failover용 legacy 조각으로 보이며, 현재 홈 UI 기준은 `Ridgeway Sans`와 `Copernicus`다.
Quick Start 01
body {
  font-family: "Ridgeway Sans", system-ui, sans-serif;
  font-weight: 300;
  line-height: 1.5;
}
Quick Start 02
:root {
  --bg: #FFF;
  --fg: #000;
  --surface: #F5F5F5;
  --surface-dark: #121212;
}
Quick Start 03
:root {
  --brand: #FA4616;
  --warning: #FEB904;
  --success: #32B67A;
  --earth: #61845B;
}
페이지 전체를 `#32B67A`나 `#61845B`로 칠하면 Patagonia처럼 보이지 않는다. 현재 홈은 검정/흰색이 1차 레이어이고, 자연색은 상태·캠페인·테마 accent다.
