Patagonia — N/A tokens

https://www.patagonia.com 이 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. 흑백 모노크롬 UI — 환경 브랜드이지만 CTA는 green/blue가 아닌 near-black(#020202).

Quick Start 5분 안에 Patagonia처럼 만들기 — 3가지만 하면 80%
1 · 폰트 body { font-family: "Avenir Next W02 Light", Helvetica, Arial, Verdana, sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #FFFFFF; --fg: #020202; /* near-black */ } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 컬러 :root { --brand: #020202; } /* CTA: bg #020202, color #FFFFFF radius: 0px text-transform: uppercase */
절대 하지 말아야 할 것 하나: 환경 브랜드라고 초록이나 파랑을 CTA에 넣는 것. Patagonia UI는 #020202 흑백 중심이다. green/blue는 제품 사진·환경 이미지에만 등장한다.

    11 · Do / Don't

    Do / Don't

    지켜야 할 것과 피해야 할 것.

    Do

    • 텍스트는 #020202(near-black)으로 설정한다
    • heading은 font-family: "Avenir Next LT W02 Bold"로 별도 지정
    • 버튼에 border-radius: 0을 유지한다
    • 환경·활동 가치 중심의 카피 톤을 유지한다
    • CTA는 #020202 흑백으로만 처리한다

    Don't

    • 초록이나 파랑을 CTA에 넣지 않는다 — UI는 흑백 중심
    • font-weight: 700으로 굵기를 제어하지 않는다 — 별도 Bold 폰트
    • 텍스트에 순흑(#000000)을 쓰지 않는다 — #020202 사용
    • 불필요한 색상 장식을 넣지 않는다
    • 큰 border-radius(8px 이상)를 버튼에 쓰지 않는다
