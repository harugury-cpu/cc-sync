DJI — .dui-btn-buy

dji.com이 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. 다크 Primary + 파란 Buy 이중 버튼 시스템이 DJI 이커머스의 핵심.

Quick Start 5분 안에 DJI처럼 만들기 — 3가지만 하면 80%
1 · 폰트 body { font-family: 'Open Sans', 'PingFang SC', Arial, sans-serif; font-weight: 400; }
2 · 다크 배경 :root { --bg: #3C3E40; --fg: #FFFFFF; } body { background: var(--bg); color: var(--fg); }
3 · 이중 버튼 /* Primary (탐색) */ .btn { bg: #3C3E40; } /* Buy (구매) */ .btn-buy { background: linear-gradient( -180deg, #1E9DF7 0%, #1392ED 100%); }
절대 하지 말아야 할 것 하나: 두 버튼 시스템을 하나로 통합하는 것. DJI는 #3C3E40 다크 회색(탐색)과 #1E9DF7 파랑(구매) 버튼을 명확히 구분한다. 이 이중 구조를 합치면 구매 전환 신호가 사라진다.

09 · Do / Don't

Do / Don't

✓ DO

  • #1E9DF7 파랑은 Buy CTA 전용
  • #3C3E40 다크는 탐색/정보 버튼
  • 두 버튼 역할 명확히 구분 유지
  • 버튼 border-radius 2px — 샤프함 필수
  • 다크 ↔ 라이트 섹션 교차
  • 기술 스펙을 정밀 수치로 — "48MP Hasselblad"

✗ DON'T

  • 두 버튼 시스템을 하나로 통합
  • 버튼에 큰 border-radius
  • Buy 버튼을 다른 색으로 교체
  • 파스텔/밝은 배경으로 전체 교체
  • all .3s ease 트랜지션 생략
오해실제설명
단색 버튼 이중 구조 다크(탐색) + 파랑(구매) 명확 구분
큰 radius 2px 직각 기술/장비 브랜드의 샤프한 엣지
라이트 테마만 다크 + 라이트 교차 드론 영상 히어로는 다크 배경이 기본
