00 · Visual Theme

분위기와 철학

실제 CSS를 열어 보면 Deloitte Digital은 corporate black/white와 campaign color surface를 전략적으로 겹친다. 얇은 Open Sans, full-screen marquee, pill CTA, green hover state가 핵심이다.

Industrial Minimalism, but not monochrome-only

`#000000`과 `#FFFFFF`가 구조를 만들고, `#D0D0CE`가 거의 모든 rail과 separator를 책임진다. 이 기본 뼈대는 매우 절제돼 있지만, 실제 인터랙션은 Deloitte green `#86BC25`가 잡는다. CTA hover, active tab, icon hover가 모두 이 green으로 연결된다.

히어로는 `.dd-marquee-carousel` 하나로 tone을 고정한다. 모바일은 `630px`, 데스크톱은 `100vh`라서 첫 화면부터 full-bleed agency mood가 나온다. headline은 `desktop:text-desktop-h2` + `.font-light`, 즉 `80px / 94px / 300` 조합으로 크게 띄우되 무겁지 않다.

또 하나 중요한 건 campaign palette다. `#007680`, `#0097A9`, `#9DD4CF`, `#A0DCFF`, `#E3E48D`가 mosaic tile과 CTA에 쓰이면서, 사이트 전체를 전형적인 회계법인 블랙박스에서 빼낸다. Deloitte green은 상태용이고, teal/sky/lime은 narrative surface용이라는 분업이 분명하다.

컴포넌트는 대부분 square surface + pill CTA 조합이다. 따라서 페이지는 단단한 직선 구조를 유지하면서도, 버튼과 일부 interactive control만 극단적으로 둥글다. 이 대비가 Deloitte Digital의 시그니처다.

02 · Tech Stack

테크 스택

구조는 AEM, 미세 조정은 Tailwind-style utility, 인터랙션은 Swiper와 component-specific state variant가 맡는다.

Runtime

Adobe Experience Manager

URL이 /etc.clientlibs/와 .aem-Grid--default--12를 사용한다. semantic block은 대부분 .cmp-* / .dd-* namespace다.
Utility Layer

Tailwind-style classes

font-light, desktop:text-desktop-h2, tracking-letter-2, transition, duration-200처럼 utility 조합이 실제 DOM에 노출된다.
Carousel

Swiper marquee

Typography

Open Sans self-host

`@font-face`로 `300 / 400 / 500 / 600 / 700 / 800` normal/italic을 모두 제공한다. 실제 tone은 `.font-light`의 `300`에서 나온다.
Theme

Mixed light/dark

light body section과 black/video hero가 공존한다. grayscale 뼈대 위에 green/teal/sky/lime이 state와 campaign layer로 얹힌다.
State

Green + gray split

#86BC25는 active/hover, #53565A는 active gray tab, #75787B는 disabled/secondary, #D0D0CE는 base rail 역할을 맡는다.

08 · Motion

모션

전반적인 motion은 과격하지 않다. utility transition이 베이스를 만들고, mosaic grid에서만 `.4s ease-out` gradient slide가 강조된다.

Utility Base

.15s cubic-bezier(.4, 0, .2, 1)

.transition가 color, border, opacity, box-shadow, transform에 같은 easing을 공유한다.
Header

duration-200

.duration-200는 sticky header translate/visibility에 쓰인다. hero 상단 nav가 과하지 않게 움직인다.
Mosaic

.4s ease-out gradient shift

.cmp-mosaic-grid__panel.with-animation가 hover에 맞춰 `background-position`을 top에서 bottom으로 이동시킨다.
Buttons

Color-only feedback

white CTA는 hover/focus/active 모두 green border/text로 바뀐다. layout shift보다 state color 전환이 중요하다.
`.cmp-mosaic-grid__cell--text-tile .cmp-mosaic-grid__panel`는 desktop `min-width:1201px`에서만 gradient slide를 켠다. 즉 Deloitte Digital의 dynamic effect는 모든 surface가 아니라 일부 showcase tile에만 집중된다.
09 · Layout Patterns

레이아웃 패턴

레이아웃은 크게 네 가지다. full-screen marquee, AEM 12-column shell, text-v2 module, hover-driven mosaic grid.

Hero

630px mobile / 100vh desktop marquee

첫 화면은 full-bleed image/video + white headline + pill CTA. `READ MORE` 버튼은 상황에 따라 white secondary 혹은 aqua primary variant를 쓴다.
Grid

AEM 12-column shell

DOM에서 `.aem-Grid--default--12`가 직접 노출된다. inner typography와 spacing은 utility layer가 덮는다.
Text Blocks

text-v2-homepage-default rhythm

홈 텍스트 블록은 mobile `pt 8 / pb 64`, desktop `pt 16 / pb 32`. `cmp-text__regular` + `font-weight__light`가 기본 텍스처다.
Interactive Surface

Mosaic grid gradient slide

black/teal/blue tile이 hover에서 gradient position을 이동시킨다. `#0097A9 -> #007680`, `#0076A8 -> #005E86`가 실제 rule로 확인된다.
Breakpoint는 `767px`, `768px`, `1024px`, `1201px`, `1400px`가 핵심이다. 특히 `1024px`에서 hero scale과 CTA padding이 크게 바뀌고, `1201px`에서 mosaic hover가 활성화된다.

12 · Do / Don't

Do / Don't

재현할 때는 thin typography, pill CTA, green/gray state split, campaign surface palette를 함께 가져가야 Deloitte Digital처럼 보인다.

Do

  • Open Sans 300을 기본 인상으로 유지한다.
  • 100vh marquee hero와 white headline 조합을 첫 화면에 쓴다.
  • CTA는 pill로 만들고 `9999px` radius와 uppercase tracking을 유지한다.
  • interaction은 green, disabled/state는 gray로 분리한다.
  • black/white base 위에 teal/sky/lime campaign surface를 남겨 둔다.

Don't

  • 순수 흑백 사이트로 평면화하지 않는다.
  • body/headline weight를 400~600 중심으로 두껍게 올리지 않는다.
  • CTA를 사각형 또는 작은 radius 버튼으로 바꾸지 않는다.
  • green hover와 gray tab state를 같은 neutral feedback으로 통일하지 않는다.
  • mosaic tile gradient를 지우고 모든 section을 같은 흰 박스로 만들지 않는다.
