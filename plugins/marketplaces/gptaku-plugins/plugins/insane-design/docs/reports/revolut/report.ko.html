Design System Report

Revolut — --rui-*

https://www.revolut.com 이 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. 모든 값은 CSS 커스텀 프로퍼티를 파싱한 결과이고, AI 추론이나 추측은 섞이지 않았다.

Quick Start 5분 안에 Revolut처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight 400 body { font-family: 'Aeonik Pro', Inter, -apple-system, sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #FFFFFF; --fg: #191C1F; } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 인디고 :root { --rui-color-accent: #4F55F1; }
절대 하지 말아야 할 것 하나: 요금제별 컬러(보라 #9539F2, 핑크 #F12587, 골드 #DD7904)를 Revolut 브랜드 컬러로 사용하는 것. #4F55F1 인디고가 canonical 브랜드 컬러다.

09 · Layout Patterns

레이아웃 패턴

Revolut.com의 핵심 레이아웃 — 플랜 카드와 Feature 섹션 중심.

Hero 섹션

  • 풀 블리드 + 텍스트 중앙, 흰 배경
  • 인디고 그라디언트 accent

  • 가로 스크롤 또는 그리드

  • Standard가 기본 강조

Feature 섹션

  • 이미지 + 텍스트 2-column
  • 번갈아 배치 — left/right alternating
  • 배경: 흰 / 연회색 교차

컨테이너 · Nav

  • max-width: 1200px; margin: 0 auto
  • Nav: sticky 흰 배경
  • Nav CTA: 인디고 버튼

13 · Do / Don't

Do / Don't

Revolut 디자인 시스템의 핵심 규칙과 자주 하는 오해.

✓ DO

  • #4F55F1 인디고를 메인 CTA와 브랜드 accent에 사용
  • Aeonik Pro로 헤드라인 — 신뢰감 있는 현대 폰트
  • 플랜 카드 상단에 플랜별 accent 컬러 바 (4px)
  • 충분한 white space — 핀테크 청결함
  • 수수료 투명성 카피 — Revolut 핵심 차별점

✗ DON'T

  • 플랜별 컬러(보라·핑크·골드)를 메인 브랜드 컬러로 혼용
  • 전통 은행 느낌의 과도한 그린 — Revolut은 인디고
  • 큰 serif 폰트 — sans-serif 현대성 필수
  • 다크 배경을 기본으로 — light 모드가 primary
  • 버튼 radius를 2px 이하로 — 핀테크는 부드러운 엣지
오해 (자주 하는 실수) 실제 이유
Premium 보라가 브랜드 #4F55F1 인디고 보라(#9539F2)는 Premium 플랜 전용 마케팅 컬러. 브랜드 canonical은 Standard 인디고.
다크 모드가 기본 라이트 모드 primary 마케팅 사이트는 흰 배경 라이트가 기본. 다크는 특정 섹션 강조용.
작은 radius 버튼 10px rounded Revolut은 핀테크이지만 친근한 현대적 radius. 2px 같은 샤프 엣지는 맞지 않음.
