Zara — --color-* 자체 토큰

패스트패션의 럭셔리 미감 — #000000 모노크롬 + Helvetica Now Text weight 300. 컬러 없음, 그림자 없음, radius 0. 흑백과 여백이 전부.

"Helvetica Now Text",

Monochrome 절제 — 패스트패션의 럭셔리 UI

흑백만으로 프리미엄 포지셔닝. weight 300 Light + radius 0 + shadow 없음.

자라(Zara)는 패스트패션 브랜드임에도 럭셔리에 근접한 극도의 절제를 UI 언어로 삼는다. 검정 #000000과 흰색 #FFFFFF의 이분법적 팔레트 위에 Helvetica Now Text 웨이트 300(Light)을 기본 body 폰트로 쓴다.

색상은 거의 쓰지 않고, 라인과 공간으로 계층을 만든다. --color-main-000(#fff)부터 --color-main-080(#333)까지 8단계 그레이 스케일이 모든 UI를 처리한다. 유일한 예외는 오류/강조 레드 #E90D01과 성공 그린 #23F444이며, 이마저도 상태 피드백 전용이다.

여백은 관대하고 리듬이 있다. 상품 이미지가 배경 역할을 하며, 텍스트 요소는 이미지 위에 최소한으로 올린다. 버튼과 카드의 radius는 0, box-shadow도 없다.

영문 텍스트는 uppercase + letter-spacing: 0.05em 조합으로 에디토리얼 감성을 만든다.

레이아웃 — 전폭 이미지 우선, 텍스트 최소화

Grid System

  • Max-width: 100% (전폭 패션 이미지)
  • Grid type: CSS Grid
  • Columns: 4열 (데스크), 2열 (모바일)

Card Patterns

  • Background: 투명 (이미지가 카드)
  • Border: none
  • Radius: 0
  • Padding: 8px 0 (이미지 아래 텍스트)
  • Shadow: 없음

Navigation

  • Type: 수평 메가메뉴
  • Height: 48px
  • Background: #FFFFFF

반복 작업 가이드
  • • 색상 변경 시: 흑백 기준. 색 추가 시 상태 컬러(#E90D01, #23F444)만.
  • • 폰트 변경 시: weight 300이 기본. 임의로 400 이상 올리지 말 것.
  • • 여백 조정 시: 각 섹션 padding은 40px~80px 수준.
  • • 새 컴포넌트: border-radius 0, border 없음 패턴.
  • • 이미지: 4:5 세로 비율 유지. object-fit: cover.
14

Do / Don't

✅ DO

  • • body font-weight 300 고정
  • • 버튼·카드 radius 0
  • • 그림자 대신 1px solid #000000
  • • 상품 이미지 aspect-ratio: 4/5
  • • 텍스트: uppercase + letter-spacing: 0.05em
  • • 색상은 흑백만. 세일만 #23F444

❌ DON'T

  • • font-weight 400 이상 body 사용 금지
  • • 버튼에 둥근 radius 추가 금지
  • • box-shadow 사용 금지
  • • 채도 있는 브랜드 컬러 추가 금지
  • • 배경에 #F5F5F5 같은 온화한 회색 금지
  • • Times New Roman을 body에 쓰지 말 것
