Purple × Yellow · Webflow 스토리텔링

Lemon Squeezy의 마케팅 사이트는 "Merchant of Record를 재미있게 판다"는 브랜드 정신을 시각화한다. 페이지는 dark #121217 (--black-2) 기반에 핵심 브랜드 purple #5423E7 (--ls-purple)가 hero·CTA·스크롤 하이라이트에 반복된다. 보색으로 yellow #FFC233이 citrus accent처럼 hero 일러스트에 등장.

색상 전략은 "purple × yellow + hot accent"의 3축. --ls-purple #5423E7 (primary) · --purple-500 #7047EB (hover tint) · yellow #FFC233 (accent) · #FFD266 (accent soft). 거기에 hot pink #DB0BB9 (--pink-700)가 한두 번 등장.

타이포그래피는 Circularpro book (Lineto, 유료) + Circularxx + Inter fallback. circular 계열의 둥근 o/a 글리프가 시그니처. 코드는 JetBrains Mono.

레이아웃은 스토리텔링형 스크롤. Webflow로 제작되어 .w-button, .w-slider-*, .w-dropdown-* util class가 DOM에 노출. max-width 1280px 기본.

인터랙션은 Webflow interactions로 scroll-driven reveal + hover scale. transition .2s ease-out.

Lemon Squeezy — --ls-* / --purple-*

Webflow 기반 토큰 시스템. purple × yellow dual-accent + dark `#121217` canvas.

/* 1. 폰트 */
body {
  font-family: "Circularpro book",
    "Circularxx",
    Inter,
    -apple-system,sans-serif;
  font-weight: 400;
}
/* 2. Dark bg + text */
:root {
  --black-2: #121217;
  --ls-white: #FFFFFF;
  --gray-500: #6C6C89;
}
body {
  background: var(--black-2);
  color: var(--ls-white);
}
/* 3. Purple × Yellow */
:root {
  --ls-purple:   #5423E7;
  --purple-500:  #7047EB;
  --ls-color-yellow: #FFC233;
  --pink-700:    #DB0BB9;
}

Do / Don't

Lemon Squeezy 규칙.

DO

  • 배경 #121217 dark primary.
  • primary CTA yellow #FFC233 + #121217 text + pill radius.
  • purple #5423E7은 secondary CTA/link/accent.
  • 본문 Circular 16px weight 400.
  • 카드 bg #1C1C24 (slightly lighter than page).

DON'T

  • purple을 Tailwind violet #6D28D9로 대체 금지.
  • yellow를 amber #F59E0B로 바꾸지 말 것.
  • body weight 500 이상 금지.
  • 카드 강한 shadow 금지.
  • monochrome dark 금지 — yellow 포인트 필수.
