Bain & Company — 다크 레드 #CC0000

https://www.bain.com 이 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. Bain의 레드는 #CC0000 — 밝은 #FF0000이 아니다.

Quick Start 5분 안에 Bain처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight 400 body { font-family: system-ui, -apple-system, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #FFFFFF; --fg: #313131; /* warm dark */ } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 컬러 :root { /* 다크 레드 — NOT #FF0000 */ --brand: #CC0000; }
절대 하지 말아야 할 것 하나: Bain red를 #FF0000(순수 빨간)으로 밝게 쓰는 것. Bain의 브랜드 red는 심오하고 절제된 다크 레드 #CC0000계열이다. 순수 빨간은 Bain 특유의 전문적 권위감을 해친다.

12 · Do / Don't

Do / Don't

Bain 디자인 시스템의 핵심 규칙.

✓ DO

  • 브랜드 red는 #CC0000(다크 레드)을 쓴다
  • primary CTA와 링크에 동일한 #CC0000을 사용한다
  • heading은 모두 font-weight: 700을 적용한다
  • 시스템 폰트를 그대로 사용한다 (외부 폰트 불필요)
  • 텍스트는 #313131(따뜻한 다크 그레이)을 사용한다

✗ DON'T

  • Bain red를 #FF0000(순수 빨간)으로 밝게 쓰지 않는다
  • 배경에 빨간 계열 색상을 쓰지 않는다 — 포인트 전용
  • font-weight: 500 이나 600을 쓰지 않는다 — 400/700만
  • 과도한 red 사용으로 UI를 채우지 않는다 — CTA·링크·태그만
  • border-radius에 큰 값(12px 이상)을 넣지 않는다
잘못된 구현
올바른 구현
이유
color: #FF0000
color: #CC0000
밝은 빨간은 Bain의 전문적 권위감을 해친다. 다크 레드가 정체성이다.
font-weight: 500
font-weight: 400 or 700
중간 weight는 사용하지 않는다. 400과 700 두 단계만.
border-radius: 12px
border-radius: 3px or 0px
Bain UI는 직각(0px) 또는 최소 라운드(3px). 큰 라운드는 정체성 훼손.
