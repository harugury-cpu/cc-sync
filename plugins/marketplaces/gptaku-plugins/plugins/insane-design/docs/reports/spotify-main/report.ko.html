Design System Report

Spotify — 실제 CSS 기반 분석

open.spotify.com의 디자인 시스템 분석 — 그린(#1ED760) Play CTA, Encore 토큰 시스템, 다크 #121212 배경의 음악 스트리밍 미학.

Quick Start 5분 안에 Spotify처럼 만들기
1 · 폰트 body { font-family: "CircularSp", "Circular", -apple-system, Helvetica, Arial, sans-serif; font-weight: 400; }
2 · 다크 배경 :root { --bg: #121212; --fg: #FFFFFF; --bg-surface: #282828; } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 그린 :root { --text-bright-accent: #1ED760; --essential-bright-accent: #1ED760; }
절대 하지 말아야 할 것 하나: 그린(#1ED760)을 배경 전면에 깔거나 모든 인터랙션 요소에 사용하는 것. Spotify 그린은 Play 버튼·Follow CTA 등 핵심 행동 유도에만 집중해야 강렬함이 살아난다. 과용하면 포인트 효과가 소멸된다.

13 · Do / Don't

Do / Don't

Spotify 미학의 핵심 원칙과 치명적 실수.

✓ DO

  • 그린(#1ED760)은 Play·Follow·CTA 핵심 행동에만 — 포인트 효과 극대화
  • 배경은 #121212 — 순검정(#000) 아닌 약간 따뜻한 near-black
  • 카드 배경은 #282828 — 배경과 레이어 구분 (중요)
  • 텍스트 subdued는 #B3B3B3 — 회색 톤으로 계층 구분
  • Play 버튼은 원형(border-radius: 50%) + 그린 글로우 효과
  • Encore 토큰명 --text-bright-accent 원본 유지 — 임의 변경 금지

✗ DON'T

  • 그린(#1ED760)을 배경 전면 사용하지 않는다 — 포인트 컬러 효과 소멸
  • 흰 배경(#FFFFFF)을 기본으로 쓰지 않는다 — Spotify는 다크 퍼스트
  • --text-bright-accent를 임의 색으로 재정의하지 않는다 — Encore 파괴
  • CircularSp weight 100으로 쓰지 않는다 — 다크 배경에서 가독성 부족
  • Orange(#FFA42B)를 primary CTA로 쓰지 않는다 — Podcast 섹션 전용
  • Play 버튼을 사각형으로 만들지 않는다 — 원형(50%)이 Spotify DNA
