DO / DON'T

DO — 이렇게 하세요

  • 배경은 #ffffff 또는 #F7F7F7 — 제품 이미지가 주인공
  • 브랜드 블루 #2189FF는 CTA 버튼, 링크, active badge에만
  • 카드 radius는 20px — 부드러운 소비자 친화 모서리
  • carousel은 --cpCarousel-* 토큰으로만 제어
  • vw 단위 사용 — 다양한 화면에서 일관 비율감
  • body text #707070 / heading #212425
  • 버튼 height 44px — 터치 타겟 최소 기준
  • 폰트 bold/700은 헤드라인과 가격에만

DON'T — 하지 마세요

  • #2189FF를 배경 면적으로 쓰지 말 것
  • 에러 레드 #FA2337을 디자인 accent로 쓰지 말 것
  • radius를 0으로 모든 요소에 적용 금지
  • 딥 네이비 #1428A0을 primary CTA에 쓰지 말 것
  • border-radius 100px 이상 사용 금지
  • 중간 회색 #707070을 헤드라인 텍스트에 쓰지 말 것
  • Swiper 없이 --swiper-* 변수 단독 참조 금지
