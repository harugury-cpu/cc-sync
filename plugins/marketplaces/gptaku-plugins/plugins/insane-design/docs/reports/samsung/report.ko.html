Design System Report

Samsung — near-black CTA

www.samsung.com/sec/ 의 실제 CSS에서 파싱한 버튼 구조·폰트·컬러 스펙. CTA는 검정, 보조 액션만 파랑 — 삼성 웹의 핵심 원칙.

Quick Start 5분 안에 Samsung처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight body { font-family: "Samsung Sharp Sans", "SamsungOneKorean", "Dotum", sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #FFFFFF; --fg: #212425; } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 컬러 :root { --brand: #212425; /* near-black = 삼성 primary */ --brand-blue: #2189FF; /* blue = 보조 액션 전용 */ }
절대 하지 말아야 할 것 하나: CTA 버튼을 파랗게 만드는 것. 삼성 primary(.btn-type2)는 #212425 — near-black이다. #2189FF 파랑은 보조 액션(.btn-type3)에만 쓴다.

09 · Do / Don't

자주 틀리는 것들

삼성 웹 구현 시 가장 많이 실수하는 패턴.

✓ DO

  • CTA 버튼은 #212425 near-black 사용
  • 한글 UI는 SamsungOneKorean 폰트 적용
  • 버튼 모서리 각지게 — border-radius: 0
  • 보조 액션에만 #2189FF 파랑 사용
  • 비활성 버튼은 #8F8F8F

✗ DON'T

  • CTA에 파란색 버튼 — 삼성 웹 primary는 검정
  • 버튼에 둥근 모서리 — 삼성 스타일과 맞지 않음
  • Samsung Sharp Sans 라이선스 없이 사용
  • #000000 순수 검정 대신 #212425 near-black 사용 필수
