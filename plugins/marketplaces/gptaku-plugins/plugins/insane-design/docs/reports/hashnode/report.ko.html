Reading-first indigo dual theme

Hashnode는 개발자가 읽는 블로그 플랫폼이라는 정체성을 시각적으로 정밀히 구현한다. 페이지는 light #FFFFFF 기반에 본문이 #101828 다크 슬레이트로 깔리고, 코드 구문 강조는 실제 GitHub dark syntax 색(#A5D6FF 문자열, #7EE787 태그/이름, #F85149 에러)을 정확히 재사용한다.

색상 전략은 brand indigo #1D52DE 단일 앵커 + dual-theme primary. light 모드에서 primary는 #1D52DE deep indigo, dark 모드에서는 #5288FF bright blue로 전환. --primary-foreground는 항상 #F5F9FF. text 계층은 #101828 → #364153 → #D1D5DC의 slate ramp.

타이포그래피는 자체 호스팅 suisseIntl (Swiss International, 유료) + suisseMono. weight 200/400/500/600을 주로 사용. 본문 16px, 기사 reading 모드 18px.

레이아웃은 reading-first. prose max-width 680–768px. Tailwind + shadcn 계열 token (--primary, --sidebar-*). Tailwind default shadow 0 1px 3px 0 #0000001A, 0 1px 2px -1px #0000001A로 subtle elevation.

인터랙션은 조용. hover color 변화 + subtle transition. Tailwind 기본.

Hashnode — --primary / shadcn

URL 하나로 추출한 shadcn/Tailwind 토큰 기반 디자인 시스템 리포트.

/* 1. 폰트 */
body {
  font-family: "suisseIntl",
    "suisseIntl Fallback",
    system-ui,
    -apple-system,sans-serif;
  font-weight: 400;
  font-size: 16px;
}
/* 2. Light default */
:root {
  --background: #FFFFFF;
  --foreground: #101828;
  --muted-foreground: #364153;
  --border: #D1D5DC;
}
/* 3. Primary indigo */
:root {
  --primary: #1D52DE;
  --primary-foreground: #F5F9FF;
}
.dark {
  --primary: #5288FF;
  --primary-foreground: #F5F9FF;
}
절대 하지 말 것: body weight를 500 또는 600으로 두지 마라. Hashnode는 읽기 플랫폼이다. 본문 400, 헤드라인 600 가중치 계단을 지켜야 읽히는 블로그 느낌이 된다.

Do / Don't

Reading-first 규칙.

DO

  • 본문 배경 #FFFFFF (light) · 텍스트 #101828.
  • primary CTA는 #1D52DE indigo, dark에서 #5288FF로 swap.
  • 본문 suisseIntl 16px weight 400.
  • 헤드라인 weight 600.
  • 코드 블록은 GitHub dark syntax 재사용 (#A5D6FF string).
  • shadcn/Tailwind token을 그대로.

DON'T

  • 본문 weight 500 또는 700 금지.
  • primary를 초록·보라로 바꾸지 말 것 — indigo 고정.
  • prose max-width 1024px 이상 금지.
  • raw hex 직접 쓰지 말 것 — shadcn token으로만.
  • 카드 shadow 너무 강하게 금지.
