Starbucks — 실제 CSS 기반 분석

starbucks.com의 디자인 시스템 분석 — 딥 그린(#006242) CTA, 따뜻한 크림 배경(#F5F1EB), SoDoSans 전용 폰트의 따뜻한 커피 브랜드 미학.

Quick Start 5분 안에 Starbucks처럼 만들기
1 · 폰트 body { font-family: "SoDoSans", "Lander Tall", "Helvetica Neue", Helvetica, Arial, sans-serif; font-weight: 400; }
2 · 크림 배경 + 포레스트 텍스트 :root { --bg: #F5F1EB; --fg: #1E3932; } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 그린 :root { --brand: #006242; --colorGreenAccent: #00754A; --forest: #1E3932; }
절대 하지 말아야 할 것 하나: 배경을 순백 #FFFFFF으로 두는 것. Starbucks 배경은 따뜻한 크림 #F5F1EB이다. 흰 배경으로 바꾸면 브랜드 따뜻함이 완전히 사라진다.

13 · Do / Don't

Do / Don't

Starbucks를 Starbucks답게 만드는 규칙.

DO

  • 배경은 크림(#F5F1EB) — 순백이 아닌 따뜻한 크림 베이스
  • 그린은 #006242 — CSS frequency 최다 chromatic, 공식 브랜드 그린
  • 카드 배경은 #FAF6EE — 크림보다 살짝 흰 warm white로 표면 구분
  • 다크 섹션은 포레스트 그린 #1E3932 — 검정 대신 그린 계열 다크
  • 제품 이미지 배경은 카라멜/골드 계열 (#F3CEA1, #CBA258)

DON'T

  • 배경을 순백(#FFFFFF)으로 쓰지 않는다 — Starbucks 브랜드 따뜻함 파괴
  • 그린 위에 에러 레드(#C82014) 쓰지 않는다 — 에러 전용 색상
  • SoDoSans weight 300으로 쓰지 않는다 — 400 이상이 기준
  • 다크 배경에 그린 CTA 그대로 쓰지 않는다 — 대비 부족, 흰 버튼 사용
  • 골드(#CBA258)를 primary CTA로 쓰지 않는다 — 포인트 · 리워드 전용
