Design System Report

HYBE Corp — --primary

hybecorp.com 실제 CSS에서 뽑은 디자인 토큰. 네온 옐로우(#F5FF00)와 짙은 검정(#111111)이 만드는 K-엔터 다크 테마.

Quick Start 5분 안에 HYBE처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight 500 body { font-family: 'Big Hit 201110', 'Noto Sans KR', sans-serif; font-weight: 500; }
2 · 다크 배경 + 텍스트 :root { --bg: #111111; --fg: #FFFFFF; } body { background: var(--bg); color: var(--fg); }
3 · 네온 브랜드 컬러 :root { --brand: #F5FF00; } a:hover { color: var(--brand); }
절대 하지 말아야 할 것 하나: 배경을 #FFFFFF로 두는 것. HYBE 메인 사이트는 다크 테마(#111111)가 기반이다. 흰 배경에 올리면 K-엔터 감성이 사라진다.

10 · Do / Don't

자주 틀리는 것들

HYBE 디자인을 구현할 때 흔히 실수하는 패턴.

✓ DO

  • 배경은 #111111 — pure black이 아닌 따뜻한 검정을 써라.
  • hover에 #F5FF00 — 네온 옐로우를 hover/accent 전용으로 정확히 써라.
  • font-weight: 700을 nav에 써라 — 굵기 대비가 핵심이다.
  • 네비게이션 텍스트에 text-transform: uppercase를 적용하라.
  • radius: 0으로 유지하라 — 각진 디자인이 HYBE 감성이다.

✗ DON'T

  • 배경을 #FFFFFF로 두지 마라 — 다크 테마 서비스다.
  • #F5FF00을 본문 텍스트에 쓰지 마라 — hover 전용이다.
  • font-weight: 400으로 두지 마라 — 기본은 500이다.
  • #007AFF를 브랜드 색으로 쓰지 마라 — Swiper 라이브러리 기본값이다.
  • 모서리를 둥글게 만들지 마라 — HYBE는 각진 디자인이다.
