# MCP 카탈로그

> 스킬 생성 시 활용 가능한 MCP 서버 목록. 전문가 에이전트가 스킬 아이디어 분석 시 적합한 MCP를 추천하는 데 사용.
> 출처: [Awesome MCP Servers 한국어 가이드](https://github.com/johunsang/vive-md) (2,596+ 서버, 32 카테고리)

## 사용법

Phase B 전문가 에이전트가 아이디어를 분석할 때, 아래 카탈로그에서 스킬에 필요한 MCP를 매칭한다.
매칭 기준: 스킬의 입력/출력/외부 연동 요구사항 → 해당 카테고리의 MCP 추천.

## 매칭 가이드

전문가 에이전트가 MCP를 추천할 때 아래 질문으로 판단:

1. 스킬이 **외부 정보**가 필요한가? → 정보 수집 카테고리
2. 스킬이 **파일을 읽거나 만드는가**? → 파일 & 문서
3. 스킬이 **데이터를 조회/분석**하는가? → 데이터 & 분석
4. 스킬이 **외부 API를 호출**하는가? → API & 자동화
5. 스킬이 **웹페이지를 조작**하는가? → 브라우저 & 웹
6. 스킬이 **알림/메시지**를 보내는가? → 커뮤니케이션
7. 스킬이 **다른 AI 모델**을 쓰는가? → AI & 모델
8. 스킬이 **비즈니스 도구**와 연동하는가? → 비즈니스 & 프로젝트

추천 시 사용자가 해당 MCP를 설치해야 함을 안내한다.
