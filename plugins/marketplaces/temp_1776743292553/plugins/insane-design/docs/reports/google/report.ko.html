Design System Report

Google — Material Design 3 (--md-ref-palette-*)

https://google.com 이 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. 모든 값은 CSS 커스텀 프로퍼티를 파싱한 결과이고, AI 추론이나 추측은 섞이지 않았다.

Quick Start 5분 안에 Google처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight body { font-family: "Google Sans Text", Arial, Helvetica, sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #FFFFFF; --fg: #202124; /* 순흑 아님 */ } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 블루 :root { --brand: #1A73E8; /* primary50 --md-ref-palette-primary50 */ }
절대 하지 말아야 할 것 하나: 텍스트를 #000000(순흑)으로 두는 것. Google UI 텍스트의 primary 값은 #202124다 — 거의 검정이지만 따뜻한 dark gray다. 순흑으로 쓰면 Google의 부드럽고 명랑한 느낌이 무너진다.

02 · Tech Stack

테크 스택

Material Design 3를 기반으로 한 Google 자체 디자인 시스템. 3-tier 토큰 계층이 특징이다.

FRAMEWORK
Google 자체 서버 렌더링 (Lit / Polymer 기반 추정)
DESIGN SYSTEM
Material Design 3 (MD3) — 토큰 prefix --md-ref-palette-*
CSS ARCHITECTURE
3-tier: ref-palette(raw hex) → sys-color(semantic) → comp(컴포넌트)
CLASS NAMING
유틸리티 + 컴포넌트 클래스 혼합 (gm3-filled-button)
DEFAULT THEME
light — bg #FFFFFF
FONT LOADING
Google Fonts CDN — Google Sans, Google Sans Text, Product Sans, Roboto Mono
CANONICAL ANCHOR
#1A73E8 — --md-ref-palette-primary50. #4285F4(primary60)와 혼동하지 않도록 주의.
03 · Typography

타이포그래피

Google Sans Text(본문)와 Google Sans(헤딩)를 구분하는 것이 핵심이다. MD3 type scale 특성상 heading은 weight 400이 많고 label/title 계열만 500이다.

Font stack
Display font Google Sans — Google 전용, Google Fonts CDN
Body font Google Sans Text — 본문용 최적화 버전
Code font Consolas, Monaco, "Courier New", monospace
Weight normal / bold 400 / 500
MD3 weight 특이사항: Google 마케팅 사이트는 일반적인 "heading = bold" 규칙을 따르지 않는다. headline과 display는 weight 400이고, label과 title 계열만 500이다. 700을 heading에 쓰면 MD3 원칙에서 벗어난다.

08 · CSS Drop-in

CSS · Drop-in

루트 스타일시트에 바로 붙여넣으면 Google 디자인 토큰을 그대로 쓸 수 있다. 

/* Google — copy into your root stylesheet */
:root {
  /* Fonts */
  --font-family: "Google Sans Text", Arial, Helvetica, sans-serif;
  --font-family-display: "Google Sans", Arial, Helvetica, sans-serif;
  --font-weight-normal: 400;
  --font-weight-bold: 500;

  /* Brand (MD3 primary ramp) */
  --brand-95: #D2E3FC;
  --brand-80: #8AB4F8;
  --brand-60: #4285F4;  /* secondary accent */
  --brand-50: #1A73E8;  /* ← canonical CTA */
  --brand-40: #1967D2;
  --brand-10: #202124;

  /* Surfaces */
  --bg-page: #FFFFFF;
  --text: #202124;  /* 순흑 아님 */
  --text-muted: #5F6368;

  /* Key spacing (MD3 4px grid) */
  --space-sm: 8px;
  --space-md: 16px;
  --space-lg: 32px;

  /* Radius */
  --radius-sm: 4px;
  --radius-card: 8px;
  --radius-btn: 20px;
}

09 · Do / Don't

Do / Don't

Google Material Design 3를 구현할 때 가장 자주 틀리는 항목들.

DO

  • 텍스트는 #202124를 써라 — 순흑이 아닌 Google의 따뜻한 dark gray다.
  • 브랜드 블루는 #1A73E8(primary50)을 써라.
  • font-weight: 500은 label/title 계열에만 써라 — 본문은 400이다.
  • MD3 4px 그리드 배수를 지켜라 (4, 8, 12, 16, 24, 32px).
  • 버튼은 border-radius: 20px(pill-ish)으로 써라.

DON'T

  • 텍스트를 #000000으로 두지 마라 — Google은 #202124를 쓴다.
  • #4285F4를 primary로 착각하지 마라 — CTA anchor는 #1A73E8이다.
  • heading에 font-weight: 700을 쓰지 마라 — MD3 headline은 400이다.
  • 버튼 radius를 0으로 두지 마라 — Google 버튼은 둥글다(20px).
  • Roboto를 기본 폰트로 쓰지 마라 — 마케팅 사이트 기본은 Google Sans Text다.
Common Mistakes
틀린 값 올바른 값 이유
#000000 #202124 Google 텍스트는 따뜻한 dark gray. 순흑을 쓰면 명랑한 Google 느낌이 사라진다.
#4285F4 #1A73E8 #4285F4는 primary60(보조 강조). CTA 버튼과 링크의 anchor는 primary50 #1A73E8이다.
font-weight: 700 (heading) font-weight: 400 MD3 headline/display는 weight 400. 700은 MD3에서 사용하지 않는다.
font-family: Roboto Google Sans Text Roboto는 Android/앱 기본 폰트. google.com 마케팅 사이트는 Google Sans Text다.
border-radius: 0 border-radius: 20px Google 버튼은 pill-ish(20px). 사각형 버튼은 MD3 원칙에서 벗어난다.
