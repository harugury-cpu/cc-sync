Design System Report

Nike — near-black + neon lime

nike.com은 순흑/순백 모노크롬 브랜드가 아니다. 실제 텍스트 컬러는 #111111(near-black)이고, AI 기능 highlight에는 #D5FF44(형광 라임 옐로)가 사용된다. Podium CDS 토큰 시스템 기반.

Quick Start 5분 안에 Nike처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight body { font-family: "Helvetica Now Text", Helvetica, Arial, sans-serif; font-weight: 400; } /* 중간 강조는 500 */ .strong { font-weight: 500; }
2 · near-black 텍스트 :root { --bg: #FFFFFF; /* ⚠️ #000000 아님! */ --fg: #111111; } body { background: var(--bg); color: var(--fg); }
3 · Neon AI highlight :root { /* AI 기능 포인트 */ --neon: #D5FF44; --neon-border: #BAD168; /* focus ring */ --focus: #1151FF; }
절대 하지 말아야 할 것: Nike를 "검정과 흰색의 단순 브랜드"로만 생각하는 것. 실제 텍스트는 #111111(near-black)이고, AI 기능에 #D5FF44 네온 포인트가 현재 아이덴티티다.

07

Do / Don't

Nike 디자인을 구현할 때 지켜야 할 핵심 규칙.

✓ DO

  • 브랜드 컬러로 #111111 (near-black, 순흑 아님)
  • AI/혁신 기능에 #D5FF44 네온 라임 포인트
  • 포커스 링은 #1151FF 블루
  • --podium-cds-color-text-primary 변수 사용
  • headline은 UPPERCASE, 짧고 임팩트 있게

✗ DON'T

  • Nike를 순흑/순백만으로 표현하지 말 것
  • 네온 라임 #D5FF44를 배경 전체에 쓰지 말 것
  • 텍스트에 순흑 #000000 대신 #111111
  • 소문자 headline 쓰지 말 것 — Nike는 UPPERCASE
