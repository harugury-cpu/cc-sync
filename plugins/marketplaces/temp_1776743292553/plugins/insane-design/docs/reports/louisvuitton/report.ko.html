Louis Vuitton — 따뜻한 다크 브라운-블랙

https://www.louisvuitton.com 이 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. Louis Vuitton Web 전용 폰트와 #19110B(따뜻한 near-black) 중심의 럭셔리 시스템.

Quick Start 5분 안에 Louis Vuitton처럼 만들기 — 3가지만 하면 80%
1 · 폰트 body { font-family: "Louis Vuitton Web", "Louis Vuitton", "Helvetica Neue", Helvetica, Arial, sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #FFFFFF; --fg: #19110B; /* warm near-black */ } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 컬러 :root { --brand: #19110B; } /* CTA: bg #19110B color #FFFFFF border-radius: 0 */
절대 하지 말아야 할 것 하나: 텍스트 컬러로 순흑(#000000)을 쓰는 것. Louis Vuitton의 주 텍스트는 따뜻한 거의-검정(#19110B)으로, 차가운 순흑과 브랜드 따뜻함이 완전히 다르다.

10 · Do / Don't

자주 틀리는 것들

Louis Vuitton 디자인 구현 시 실제 CSS에서 확인된 규칙.

✓ DO

  • 텍스트 컬러는 #19110B(따뜻한 다크)을 쓴다
  • 버튼과 카드에 border-radius: 0을 유지한다
  • 폰트는 weight 단일(400)로 사용한다
  • 버튼 uppercase에 letter-spacing: 0.12em 적용
  • 여백을 80px 이상으로 넉넉하게 유지한다

✗ DON'T

  • 텍스트에 순흑 #000000을 쓰지 않는다 — #19110B로 대체
  • border-radius에 큰 값을 넣지 않는다
  • 채도 높은 악센트 색상을 UI에 추가하지 않는다
  • body에 font-weight: 700을 쓰지 않는다
  • hover 시 색상 변화 대신 언더라인·불투명도로만 처리
