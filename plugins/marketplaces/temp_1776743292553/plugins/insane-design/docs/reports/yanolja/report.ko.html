Design System Report

Yanolja — Tailwind v4

yanolja.com 실제 CSS에서 뽑은 디자인 토큰. Pretendard + 다층 블루 팔레트(#4154FF primary main)로 구성된 여행 플랫폼 디자인 시스템.

Quick Start 5분 안에 Yanolja처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight body { font-family: Pretendard, 'Apple SD Gothic Neo', Roboto, Arial, sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #FFFFFF; --fg: #000000; } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 컬러 :root { --brand: #4154FF; } /* primary main CTA */
절대 하지 말아야 할 것 하나: 브랜드 블루를 #3C7BFD 하나로 고정하는 것. Yanolja CTA는 #4154FF(primary main)이고, 그라디언트/버블 색은 각각 다른 블루 계열이 혼용된다.

08 · Do / Don't

자주 틀리는 것들

Yanolja 디자인을 구현할 때 흔히 실수하는 패턴.

✓ DO

  • CTA 버튼은 #4154FF(primary main)을 정확히 써라.
  • 버블 배경 #F5F6FF, 테두리 #B2BAFF로 쌍으로 써라.
  • Pretendard weight는 400(본문), 700(강조)으로 명확히 구분하라.
  • 그라디언트는 #3C7BFD → #643EDF 방향으로 써라.
  • focus outline에 #4154FF를 써라.

✗ DON'T

  • 단일 블루 하나로 모든 색을 통일하지 마라 — Yanolja는 블루 계층 시스템이다.
  • #B620E0(보라)를 primary로 착각하지 마라 — accent 중 하나다.
  • #000000을 버튼 배경으로 쓰지 마라 — 텍스트 기본값이다.
  • font-weight: 600을 기본으로 두지 마라 — Yanolja 본문은 400이다.
  • 임의 hex를 Tailwind 클래스에 하드코딩하지 마라.
