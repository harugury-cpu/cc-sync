Design System Report

Dyson — --yxt-*

dyson.com 이 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. 이커머스 프리미엄 하드웨어 브랜드의 클린 화이트 + 파란 CTA 시스템.

Quick Start 5분 안에 Dyson처럼 만들기 — 3가지만 하면 80%
1 · 폰트 body { font-family: system-ui, "Segoe UI", Roboto, sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #FFFFFF; --fg: #212121; } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 파랑 :root { --brand: #0F70F0; } .btn-primary { background: var(--brand); color: #fff; border-radius: 4px; }
절대 하지 말아야 할 것 하나: CTA 버튼에 background: #EFEFEF 같은 회색 배경을 쓰는 것. Dyson CTA의 정체성은 #0F70F0 파란색 단 하나다. 회색 CTA는 프리미엄 이커머스가 아닌 평범한 폼 UI처럼 보인다.

10 · Do / Don't

Do / Don't

Dyson 디자인 시스템을 복제할 때 가장 중요한 규칙들.

✓ DO

  • CTA 버튼에 #0F70F0 파란색 사용
  • 배경은 #FFFFFF 또는 #F5F5F5 — 흰 공간이 프리미엄감
  • 텍스트는 #212121에서 #555555 범위 내로
  • 버튼 radius는 4px–6px 미니멀하게
  • 제품 이미지 품질에 집중 — 이미지가 곧 디자인
  • 기술 스펙 수치를 카피에 직접 사용 (신뢰도)

✗ DON'T

  • CTA에 #EFEFEF 회색 배경 — 파란색만이 Dyson CTA
  • 화려한 그라디언트 배경 — Dyson은 클린 화이트
  • 무리하게 웹폰트 추가 — 시스템 폰트가 전략적 선택
  • 낮은 대비 컬러로 텍스트 — 프리미엄 = 가독성
  • 다수의 포인트 컬러 — 파랑 하나로 충분
  • 다크 배경을 전체 페이지에 — 히어로 섹션 한정
오해실제설명
#000000 검은 배경 #FFFFFF 흰 배경 마케팅 사이트 기본은 흰 배경. 다크는 특정 히어로에만
커스텀 웹폰트 system-ui 스택 글로벌 이커머스 — 웹폰트 없이 시스템 폰트로 속도 우선
빨간/노란 포인트 #0F70F0 단색 Dyson 디지털 UI는 파란 단색 CTA 전략
