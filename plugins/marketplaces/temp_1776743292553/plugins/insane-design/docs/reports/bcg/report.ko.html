Design System Report

BCG — --BCG-* · 딥 포레스트 그린

https://www.bcg.com 이 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. 브랜드 primary는 딥 포레스트 그린 #0E3E1B — 라임 그린 #7EF473과 혼동하지 말 것.

Quick Start 5분 안에 BCG처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight 300 body { font-family: "henderson-bcg-sans", "Helvetica Neue", Arial, sans-serif; font-weight: 300; /* ⚠️ 400 아님 */ }
2 · 배경 + 텍스트 :root { --bg: #FFFFFF; --fg: #212427; /* cool dark */ } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 컬러 :root { /* 딥 포레스트 그린 */ --brand: #0E3E1B; /* NOT #7EF473 (라임 — 악센트) */ }
절대 하지 말아야 할 것 하나: 브랜드 green을 #7EF473(lime green)으로 착각하는 것. BCG의 primary brand color는 딥 포레스트 그린 #0E3E1B이고, #7EF473은 검색 필터 버튼 같은 UI 악센트에만 쓰인다. 또한 body weight를 400으로 두면 BCG의 섬세한 타이포그래피 정체성이 사라진다.

12 · Do / Don't

Do / Don't

BCG 디자인 시스템의 핵심 규칙.

✓ DO

  • body weight는 300을 쓴다 — BCG 타이포그래피의 핵심
  • 브랜드 green은 #0E3E1B(딥 포레스트)를 primary로 쓴다
  • #7EF473(lime green)은 UI 악센트(검색 필터 등)에만 쓴다
  • heading은 font-family: henderson-bcg-headline, weight 700
  • 텍스트 컬러는 #212427(cool dark)을 사용한다

✗ DON'T

  • #7EF473을 브랜드 primary color로 쓰지 않는다 — 악센트 전용
  • body weight를 400으로 두지 않는다 — BCG body는 300
  • #D82216(red)을 브랜드 컬러로 쓰지 않는다 — 에러 전용
  • Video.js의 #73859F를 UI 색상으로 차용하지 않는다 — 라이브러리 전용
  • 과도한 green 사용으로 UI를 채우지 않는다 — green은 포인트만
