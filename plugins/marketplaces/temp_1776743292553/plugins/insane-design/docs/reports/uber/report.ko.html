Design System Report

Uber — Base Web (no CSS custom props)

https://uber.com 이 실제로 서빙하는 CSS 에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. CSS 커스텀 프로퍼티 없이 직접 hex 값으로 운용하는 Base Web 인라인 스타일 구조다.

Quick Start 5분 안에 Uber처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight 500 body { font-family: UberMoveText, system-ui, "Helvetica Neue", Helvetica, Arial, sans-serif; font-weight: 500; /* ⚠️ 400 아님. Uber body는 500 */ }
2 · 배경 + 텍스트 :root { --bg: #FFFFFF; --fg: #000000; } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 컬러 :root { --brand: #276EF1; /* Uber blue, 40 hits in DOM */ } /* CTA 버튼 */ .btn-primary { background: #276EF1; font-family: UberMove; font-weight: 700; }
절대 하지 말아야 할 것 하나: UberMoveText와 UberMove를 혼용하는 것. UberMoveText(count: 53)는 본문용, UberMove(count: 34)는 display/heading용이다. 둘을 구분 없이 쓰면 타이포 위계가 무너진다.

09 · Do / Don't

Do / Don't

가장 흔한 실수와 올바른 패턴. Uber를 Uber답게 만드는 규칙이다.

DO

  • UberMoveText는 본문에, UberMove는 heading에 써라 — 둘을 구분하라.
  • 본문 weight는 500을 써라 — Uber는 400이 아닌 500이 기본이다.
  • 브랜드 블루는 #276EF1을 써라.
  • 배경 계층을 지켜라 — #FFFFFF(main), #F3F3F3(secondary), #E8E8E8(구분선).
  • 테두리는 #E8E8E8(soft) 또는 #E2E2E2(standard)를 써라.

DON'T

  • UberMoveText를 heading에 쓰지 마라 — heading은 UberMove다.
  • font-weight: 400을 기본으로 두지 마라 — Uber는 500이다.
  • #000000 순흑을 배경으로 쓰지 마라 — Uber는 라이트 테마다.
  • #ADDEC9(녹색)를 CTA에 쓰지 마라 — 드라이버 상태 표시 accent다.
  • 배경에 #E8E8E8을 쓰지 마라 — 그건 테두리/구분선 색이다.
