Volvo — font-weight 9종, OneTrust 컬러

volvo.com CSS 파싱의 두 가지 발견: (1) 컬러 데이터 전체가 OneTrust 쿠키 동의 UI — 실제 Volvo 브랜드 블루는 미추출. (2) Volvo Novum은 weight 100~bold까지 9가지 변형. 이 타이포그래피 다양성이 스칸디나비아 디자인의 핵심이다.

Quick Start 5분 안에 Volvo처럼 만들기 — 3가지만 하면 80%
1 · 폰트 — 다중 weight body { font-family: "Volvo Novum", "Arial Narrow", "Helvetica Neue", Arial, sans-serif; font-weight: 400; } h1 { font-weight: 300; } .caption { font-weight: 100; }
2 · 배경 + 텍스트 :root { --bg: #FFFFFF; --fg: #1F1F1F; } body { background: var(--bg); color: var(--fg); }
3 · 다크 버튼 :root { /* OneTrust 레이어 파싱 */ --btn-dark: #1F1F1F; --btn-hover: #3A3A3A; --btn-active: #4C4C4C; }
절대 하지 말아야 할 것: 폰트를 일반 Regular(400)만 쓰는 것. Volvo Novum은 SemiLight 변형이 별도로 있고, weight spectrum이 100~bold까지 9가지. 이 다양한 weight 조합이 Volvo 타이포그래피의 핵심이다.

01

출처와 수집 기준

CSS 커스텀 프로퍼티를 직접 파싱했으나 컬러 데이터 대부분이 OneTrust 쿠키 동의 UI다.

Source URLhttps://volvo.com
Fetched2026-04-13
Token prefix--ot-btn-* (OneTrust 동의 버튼 레이어)
MethodCSS 커스텀 프로퍼티 직접 파싱 · AI 추론 없음
한계실제 Volvo 브랜드 CSS는 정적 파싱으로 추출 불충분
OneTrust 경고: 파싱된 컬러 토큰(--ot-btn-primary-bg 등)은 Volvo 마케팅 사이트가 아닌 OneTrust 쿠키 동의 UI의 것이다. 실제 Volvo 브랜드 블루는 CSS에서 추출되지 않았다. #1C6BBA 같은 Volvo 브랜드 블루를 이 데이터에서 유추하지 말 것.
02

타이포그래피

Volvo Novum — 스칸디나비아 감성의 narrow sans. 9가지 weight 변형이 핵심이다.

03

컬러 (OneTrust 레이어)

파싱된 컬러는 OneTrust 쿠키 동의 UI의 것이다. Volvo 브랜드 CSS는 미추출.

OneTrust 버튼 — Dark / Light 테마
Dark Btn BG#1F1F1F
Dark Hover#3A3A3A
Dark Active#4C4C4C
Light Btn BG#E5E7EB
주의: 이 컬러들은 OneTrust 쿠키 동의 UI의 것이다. Volvo 자체 브랜드 컬러(파란 계열, 스칸디나비아 컬러)는 CSS 파싱에서 추출되지 않았다. #1C6BBA 등의 Volvo 브랜드 블루는 이 데이터와 무관하다.
04

컴포넌트

OneTrust 레이어에서 파싱된 버튼 스타일. 다크 배경 + 흰 텍스트 패턴.

버튼 스타일 (OneTrust 파싱값): background:#1F1F1F; color:#FFFFFF; border:1px solid #FFFFFF; hover:background:#3A3A3A; active:background:#4C4C4C — 이 값은 Volvo 실제 버튼이 아닌 OneTrust 레이어에서 파싱됨.
05

CSS 스니펫

루트 스타일시트에 복사해서 바로 쓸 수 있는 Volvo 디자인 토큰.

/* Volvo — copy into your root stylesheet */
:root {
  /* Fonts */
  --volvo-font-family: "Volvo Novum", "Arial Narrow",
    "Helvetica Neue", Arial, sans-serif;
  --volvo-font-family-semi: "Volvo Novum SemiLight",
    "Arial Narrow", "Helvetica Neue", Arial, sans-serif;
  --volvo-font-weight-normal: 400;
  --volvo-font-weight-bold: 700;

  /* Brand (다크 버튼 / OneTrust 레이어에서 관찰) */
  --volvo-color-btn-dark: #1F1F1F;
  --volvo-color-btn-dark-hover: #3A3A3A;
  --volvo-color-btn-dark-active: #4C4C4C;

  /* Surfaces */
  --volvo-bg-page: #FFFFFF;
  --volvo-bg-dark: #1F1F1F;
  --volvo-text: #1F1F1F;
  --volvo-text-muted: #4C4C4C;
}

/* Typography weight scale */
.text-100 { font-weight: 100; }
.text-300 { font-weight: 300; }
.text-400 { font-weight: 400; }
.text-500 { font-weight: 500; }
.text-600 { font-weight: 600; }
.text-700 { font-weight: 700; }
06

Do / Don't

Volvo 디자인을 구현할 때 지켜야 할 핵심 규칙.

✓ DO

  • 폰트 weight를 다양하게 (100~bold 9단계)
  • Volvo Novum SemiLight 변형 활용
  • 버튼은 다크 배경 #1F1F1F + 흰 텍스트
  • 스칸디나비아식 미니멀 레이아웃 (여백 충분)
  • 안전과 지속가능성 메시지 강조

✗ DON'T

  • weight를 400 하나로만 쓰지 말 것
  • OneTrust 색(#1C6BBA 등)을 Volvo 브랜드 색으로 쓰지 말 것
  • CSS 파싱에서 미확인된 Volvo 브랜드 블루를 추정으로 채우지 말 것
  • 레이아웃을 복잡하게 만들지 말 것
