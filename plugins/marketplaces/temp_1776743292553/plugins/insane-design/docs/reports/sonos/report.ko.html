Sonos — --color-accentRed

sonos.com이 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. 로즈 테라코타 #DE8579 + aktiv-grotesk 조합이 하이엔드 오디오 브랜드를 만든다.

Quick Start 5분 안에 Sonos처럼 만들기 — 3가지만 하면 80%
1 · 폰트 body { font-family: "aktiv-grotesk", Helvetica, Arial, sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #FFFFFF; --fg: #2E2E2E; } body { background: var(--bg); color: var(--fg); }
3 · 테라코타 액센트 :root { --accent-red: #DE8579; --accent-gold: #C59C6E; }
절대 하지 말아야 할 것 하나: #DE8579를 일반 오렌지/레드로 대체하는 것. 이 색은 로즈 테라코타 — 차갑지도 뜨겁지도 않은 절제된 따뜻함이다. 일반 오렌지를 쓰면 하이엔드 오디오가 아닌 소비재 브랜드처럼 보인다.

10 · Do / Don't

Do / Don't

Sonos 디자인을 복제할 때 가장 중요한 규칙들.

✓ DO

  • #DE8579 테라코타를 CTA에 — Sonos의 따뜻함
  • #C59C6E 골드를 프리미엄 배지에 활용
  • 흰 배경 + 넉넉한 여백 — 하이엔드 신호
  • 제품 사진을 크고 선명하게
  • aktiv-grotesk 또는 Inter 사용
  • 경험 중심 카피 — 스펙보다 소리 경험

✗ DON'T

  • 일반 오렌지로 #DE8579 대체 — 전혀 다름
  • 배경에 강한 색상 — 흰 배경이 프리미엄
  • 3가지 이상 포인트 컬러 — 테라코타+골드면 충분
  • 작은 제품 이미지 — Sonos는 제품이 주인공
  • 과도한 그림자 — 미니멀 elevation
오해실제설명
검정 브랜드 #DE8579 테라코타 검정은 텍스트. 브랜드 액센트는 로즈 테라코타
오렌지 대체 절제된 핑크-레드 #DE8579는 채도 낮은 테라코타 — 오렌지와 다름
무채색 브랜드 테라코타 + 골드 2색 따뜻한 액센트 시스템이 Sonos 온기
