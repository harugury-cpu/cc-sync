Design System Report

Coinbase — Coinbase Font Family

https://www.coinbase.com 이 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. OneTrust SDK CSS와 Coinbase React/Next.js 번들 모두 분석. 모든 값은 실제 CSS 파싱 결과이며, AI 추론이나 추측은 섞이지 않았다.

Quick Start 5분 안에 Coinbase처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight 400 body { font-family: CoinbaseSans, -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #FFFFFF; --fg: #0A0B0D; } body { background: var(--bg); color: var(--fg); }
3 · 로얄 블루 :root { --brand: #0052FF; } /* CTA, 링크 전용 */
절대 하지 말아야 할 것 하나: #0052FF를 일반 파랑으로 대체하는 것. 이 값은 Coinbase 고유의 진한 로얄 블루로, #0066CC 같은 소프트 블루나 #1D9BF0 트위터 블루와 전혀 다른 포지셔닝이다. OneTrust SDK 설정에서도 Coinbase가 직접 이 값을 지정해 CTA 버튼과 링크에 사용한다.

13 · Do / Don't

Do / Don't

Coinbase 디자인 시스템을 올바르게 구현하기 위한 핵심 원칙.

DO

  • #0052FF 로얄 블루를 메인 CTA에 — Coinbase 신뢰 컬러
  • CoinbaseSans로 UI 텍스트, CoinbaseDisplay로 히어로
  • 가격 표시에 CoinbaseCondensed — 숫자 가독성
  • 지갑 주소에 CoinbaseMono — 무결성 표시
  • 흰 배경 + 블루 CTA — 접근성과 신뢰 동시 전달
  • 버튼 radius 8px — 중립적 핀테크 스타일

DON'T

  • 소프트 블루(#0066CC 등)로 #0052FF 대체 — 로얄 블루 강도가 중요
  • 다크 배경을 기본으로 — 라이트가 Coinbase의 주 테마
  • 가격에 serif 폰트 — Condensed mono 계열 유지
  • 지갑 주소에 일반 폰트 — CoinbaseMono 전용
  • CSS 커스텀 프로퍼티명 발명 — Coinbase는 token 미사용
  • CoinbaseIcons를 브랜드 폰트로 착각 — 아이콘 전용
오해 실제 설명
Coinbase = 크립토 = 어둡고 화려함 #FFFFFF 라이트 라이트 테마 + 로얄 블루 — 신뢰 우선. 화려한 다크 UI는 Coinbase가 아니다.
CSS --token 기반 DS 커스텀 프로퍼티 미사용 자체 Coinbase 폰트 패밀리 기반 DS. CSS 변수 없이 직접 hex값 사용.
일반 파랑으로 대체 가능 #0052FF 고유 로얄 블루 #0052FF는 Coinbase 고유 진한 로얄 블루. 소프트 블루로 대체 시 브랜드 포지셔닝 훼손.
OneTrust CSS = 노이즈 브랜드 컬러 확인 근거 Coinbase가 OneTrust를 #0052FF로 직접 설정 — 브랜드 컬러의 canonical 확인.
