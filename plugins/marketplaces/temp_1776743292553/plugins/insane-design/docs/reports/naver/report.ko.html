Design System Report

Naver — 시스템 폰트 · 카테고리 컬러 5종

www.naver.com 의 CSS. React CSS Modules + --color_* CSS 변수. #03C75A 네이버 그린은 공식 브랜드색 (CSS 직접 미확인). 카테고리별 필터 색 5종 별도.

Quick Start5분 안에 Naver처럼 만들기
1 · 시스템 폰트body { font-family: -apple-system, "Malgun Gothic", "맑은 고딕", "Apple SD Gothic Neo", sans-serif; font-weight: 400; }
2 · 배경 + 텍스트:root { --bg: #FFFFFF; --fg: #101010; } body { background:var(--bg); color:var(--fg); }
3 · 브랜드 그린:root { --brand: #03C75A; /* 카테고리 필터 */ --enter: #E65DA0; --sports: #0147B5; }
절대 하지 말아야 할 것 하나: 카테고리 필터 색 #E65DA0(엔터), #0147B5(스포츠), #EBAA00(레시피), #008F76(금융)을 네이버 브랜드 색으로 혼동. 이들은 섹션별 필터 버튼 전용색이며, 네이버 브랜드 그린은 #03C75A다.

03 · Typography

시스템 폰트 스택 — 전용 폰트 없음

네이버는 별도 전용 폰트 없이 OS 시스템 폰트만 사용. weight는 400~800 다양하게 분포.

Font stack
System stack-apple-system, BlinkMacSystemFont, "Malgun Gothic", "맑은 고딕", helvetica, "Apple SD Gothic Neo", sans-serif
Weight normal400
Weight bold700 (헤드라인, 강조)
Weight range400 / 500 / 600 / 700 / 800 — 다양한 굵기 혼용
주의: NanumSquare는 네이버 기본 폰트가 아님 — 일부 위젯 전용. 기본값은 시스템 폰트 스택.

08 · Do / Don't

자주 틀리는 것들

Naver 웹 구현 시 가장 많이 실수하는 패턴.

✓ DO

  • 시스템 폰트 스택 사용 — 전용 폰트 없음
  • --color_* CSS 변수로 테마 대응
  • 카테고리별 고유 색상 유지 (엔터=핑크, 스포츠=파랑 등)
  • html[data-theme=greenDark] 다크 모드 분기 처리

✗ DON'T

  • 카테고리 필터 색 #E65DA0, #0147B5를 네이버 브랜드 색으로 사용
  • 하드코딩 hex 직접 사용 — --color_* CSS 변수 유지
  • NanumSquare를 네이버 기본 폰트로 설정 — 일부 위젯 전용
  • 브랜드 그린 #03C75A를 카테고리 필터에 사용 — 녹색은 금융 섹션만
