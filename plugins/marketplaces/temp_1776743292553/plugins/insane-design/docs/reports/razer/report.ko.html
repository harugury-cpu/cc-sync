Design System Report

Razer — --primary, --cx-*

https://www.razer.com 이 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. 모든 값은 CSS 커스텀 프로퍼티를 파싱한 결과이고, AI 추론이나 추측은 섞이지 않았다.

Quick Start 5분 안에 Razer처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight 400 body { font-family: RazerF5, "Helvetica Neue", Arial, "Noto Sans", sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #000000; --fg: #FFFFFF; } body { background: var(--bg); color: var(--fg); }
3 · 레이저 그린 :root { --primary: #44D62C; --cx-g-color-primary:#44D62C; --cx-color-primary: #44D62C; }
절대 하지 말아야 할 것 하나: 레이저 그린을 다른 그린으로 대체하는 것. #44D62C는 전기 네온 그린 — 라임·민트·에메랄드 그린과 전혀 다르다. 이 미묘한 차이가 게이밍 브랜드 정체성의 전부다.

04 · Colors

컬러

CSS 커스텀 프로퍼티에서 직접 파싱한 값만 수록. #44D62C 레이저 그린은 세 토큰이 동일 값을 가리킨다.

Razer Brand Green — 세 토큰 모두 동일
--primary #44D62C
--cx-g-color-primary #44D62C
--cx-color-primary #44D62C
SAP CX 컨텍스트 컬러 — e-commerce 기능 UI 전용
--cx-color-primary (dark) #6BD3FF
--cx-color-primary (light) #004CCB
--cx-color-primary (medium) #1F7BC0
Neutrals
기본 배경 #000000
메가메뉴 링크 #BBBBBB
보조 텍스트·링크 #EEEEEE
액티브 텍스트 #FFFFFF
레이저 그린은 대체 불가: #44D62C는 전기 네온 그린이다 — 일반 #00FF00보다 약간 어둡고 황색 기운이 있다. 라임·민트·에메랄드 그린으로 대체하면 브랜드 정체성이 완전히 무너진다. 검정 배경 위에서만 이 컬러가 진짜 빛난다.

09 · Layout Patterns

레이아웃 패턴

Razer.com의 핵심 레이아웃 패턴 — 풀 블리드 제품 이미지와 메가 메뉴가 중심.

Hero 섹션

  • 풀 블리드 제품 이미지, 검정 배경
  • 레이저 그린 텍스트 포인트
  • 대문자 헤드라인 — "ULTIMATE PERFORMANCE."

메가 메뉴

  • razer-mega-menu 클래스
  • 검정 배경 + #BBB 링크 텍스트
  • 활성 링크는 #44D62C 그린

제품 그리드

  • 다크 카드, 제품 이미지 중심
  • 그린 글로우 card hover 효과
  • 가격·스펙 간결 표시

컨테이너 · Nav

  • max-width: 1280px; margin: 0 auto
  • Nav: 투명 → 스크롤 시 검정 불투명

13 · Do / Don't

Do / Don't

Razer 디자인 시스템의 핵심 규칙과 자주 하는 오해.

✓ DO

  • #44D62C 레이저 그린을 액센트로 사용 — 이 컬러가 전부
  • 버튼 텍스트는 #000000 — 그린 배경 위 검정 필수
  • RazerF5 폰트로 헤드라인 — 브랜드 정체성
  • 그린 글로우 이펙트: rgba(68,214,44,.4) box-shadow
  • 대문자 텍스트 "SHOP NOW" — 게이밍 컨벤션
  • 검정 배경에 그린 포인트 — 절대 반전 없음

✗ DON'T

  • 라임·민트·에메랄드로 #44D62C 대체 — 전혀 다른 컬러
  • 흰 배경에 레이저 그린 — 다크 배경이 필수 컨텍스트
  • 버튼 텍스트를 흰색으로 — 그린 배경 위는 검정
  • 큰 border-radius — 게이밍은 샤프 엣지
  • 소문자 카피 — Razer는 대문자 언어
  • SAP CX 파란 토큰을 브랜드 컬러로 혼동
오해 (자주 하는 실수) 실제 이유
#00FF00 순수 그린 #44D62C 레이저 그린은 약간 어둡고 황색 기운 있는 전기 네온. 순수 그린과 다르다.
라이트 테마 존재 순수 다크 마케팅 사이트는 검정 배경 100%. 라이트 테마는 없다.
버튼 텍스트 흰색 #000000 그린 배경(#44D62C) 위에는 검정 텍스트. 흰색은 대비 역할 혼동.
#6BD3FF 브랜드 파랑 SAP CX 토큰 --cx-color-primary dark 컨텍스트 값 — SAP e-commerce UI 전용. 브랜드 컬러 아님.
