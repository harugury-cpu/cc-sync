Design System Report

Airbnb — --palette-*

https://airbnb.com 이 실제로 서빙하는 CSS 에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. 토큰 prefix는 --palette-*. 모든 값은 CSS 커스텀 프로퍼티를 파싱한 결과이며, AI 추론이나 추측은 섞이지 않았다.

Quick Start 5분 안에 Airbnb처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight 400 body { font-family: 'Airbnb Cereal VF', 'Circular', -apple-system, BlinkMacSystemFont, Roboto, 'Helvetica Neue', sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #FFFFFF; --fg: #222222; } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 컬러 :root { --brand: #FF385C; /* --palette-bg-primary-core */ } /* CTA 버튼 */ .btn-primary { background: #FF385C; border-radius: 8px; }
절대 하지 말아야 할 것 하나: 모든 텍스트를 font-weight: 600으로 통일하는 것. Airbnb는 weight 체계가 매우 세밀하다 — 본문 400, 강조 500, 버튼 600, 큰 제목 800까지 층위가 명확하다. 전부 600으로 뭉개면 Airbnb 특유의 위계가 사라진다.

03 · Typography

타이포그래피

Airbnb Cereal VF 가변 폰트 스택, 4단계 weight 정의, 8-step 타이포그래피 스케일과 negative letter-spacing 규칙.

Font stack
Display font Airbnb Cereal VF (Airbnb 전용 가변 폰트)
Fallback Circular (유료) → -apple-system → BlinkMacSystemFont → Roboto → Helvetica Neue
Code font monospace
Weight normal / bold 400 (normal) · 500 (medium) · 600 (semibold) · 800 (bold)
라이선스 주의: Airbnb Cereal VF는 Airbnb 전용 폰트라 재배포 불가. 자체 프로젝트에서는 Circular(유료) 또는 Nunito(무료)가 근접한 대체재다.

Negative letter-spacing이 Airbnb 타이포의 핵심: 작은 본문(text-micro~text-regular)은 양수 tracking(+0.1~+0.3px), 큰 제목(title-3~title-1)은 음수 tracking(-0.4px → -0.8px)으로 간다. 제목이 커질수록 글자 간격을 더 조여야 한다.

10 · Do / Don't

Do / Don't

가장 흔한 실수와 올바른 패턴. Airbnb를 Airbnb답게 만드는 규칙이다.

DO

  • 브랜드 CTA 버튼은 #FF385C(--palette-bg-primary-core)를 써라.
  • 텍스트는 #222222를 써라 — 순흑이 아닌 Airbnb의 따뜻한 dark gray다.
  • weight 층위를 지켜라 — 400(본문), 500(강조), 600(버튼), 800(제목).
  • 큰 제목에 negative letter-spacing을 써라 — -0.4px부터 -0.8px까지.
  • Airbnb Cereal VF가 없으면 Circular를 fallback으로 지정하라.

DON'T

  • font-weight: 600으로 모든 텍스트를 통일하지 마라 — weight 층위가 무너진다.
  • 배경을 #F7F7F7로 두지 마라 — 그건 hover/secondary 배경이고, 기본은 #FFFFFF다.
  • #FF0000이나 #FF4466을 brand red로 쓰지 마라 — 정확히 #FF385C를 써야 한다.
  • Luxe variant(#460479)를 일반 CTA에 쓰지 마라 — Airbnb Luxe 서비스 전용이다.
  • 버튼 radius를 0으로 두지 마라 — Airbnb 버튼은 8px이다.

올바른 값

Airbnb의 타이포 위계는 4단계 weight 층위로 만들어진다. 600으로 통일하면 아무 것도 강조되지 않는다.

letter-spacing: 0 (제목)
