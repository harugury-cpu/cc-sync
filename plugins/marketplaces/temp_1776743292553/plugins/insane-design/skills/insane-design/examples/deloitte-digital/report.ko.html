Design System Report

Deloitte Digital — Open Sans Light · near-white

https://www.deloittedigital.com 이 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. body weight 300(Light)과 배경 #FBFBFB(near-white)가 Deloitte Digital의 핵심 정체성이다.

Quick Start 5분 안에 Deloitte Digital처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight 300 body { font-family: "Open Sans", Calibri, sans-serif; font-weight: 300; /* ⚠️ 400 아님 */ }
2 · 배경 + 텍스트 :root { /* near-white, NOT #FFFFFF */ --bg: #FBFBFB; --fg: #000000; } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 컬러 :root { /* 흑백 중심 */ --brand: #000000; /* Deloitte 녹색 UI 사용 금지 */ }
절대 하지 말아야 할 것 둘: (1) 배경을 순백(#FFFFFF)으로 두는 것 — Deloitte Digital 배경은 #FBFBFB(극도로 미세하게 따뜻한 near-white)이다. (2) body weight를 400으로 설정하는 것 — 300(Light)이 정체성이다. 400으로 두면 전체 느낌이 순식간에 평범해진다.

12 · Do / Don't

Do / Don't

Deloitte Digital 디자인 시스템의 핵심 규칙.

✓ DO

  • body weight는 300(Open Sans Light)을 쓴다
  • 배경은 #FBFBFB(near-white)로 설정한다 — 순백이 아님
  • 경계선은 #DCDCDC를 사용한다
  • heading은 font-weight: 700으로 대비를 준다
  • Open Sans는 300 weight 전용으로 로딩한다

✗ DON'T

  • body weight를 400으로 두지 않는다 — 300이 정체성
  • 배경을 순백(#FFFFFF)으로 두지 않는다 — #FBFBFB 사용
  • font-weight: 500, 600을 쓰지 않는다
  • 브랜드 컬러로 Deloitte 녹색을 UI에 넣지 않는다 — 흑백 중심
  • border-radius에 16px 이상 큰 값을 넣지 않는다
