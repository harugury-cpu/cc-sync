Hyundai — Element UI + 현대 커스텀

www.hyundai.com/kr/ko/ 의 CSS. Vue.js + Element UI 기반. 브랜드 액티브 색은 #007FA8 — Element UI 기본 파랑(#409EFF)과 다르다.

Quick Start 5분 안에 Hyundai처럼 만들기
1 · 폰트 body { font-family: "HyundaiSansTextKR", sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #FFFFFF; --fg: #303133; } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 :root { --brand: #007FA8; /* 현대 teal — active */ /* #409EFF = Element UI 기본 */ }
절대 하지 말아야 할 것 하나: #409EFF를 현대 브랜드 색으로 사용하는 것. 이 파랑은 Element UI 라이브러리 기본값이다. 현대 브랜드 액티브는 #007FA8 teal.

09 · Do / Don't

자주 틀리는 것들

현대 웹 구현 시 가장 많이 실수하는 패턴.

✓ DO

  • 브랜드 액티브 색 #007FA8 사용
  • 헤드라인 HyundaiSansHeadKR, 본문 HyundaiSansTextKR 구분
  • 중립 텍스트 #303133 (Element UI 기본)
  • Element UI 컴포넌트 패턴 유지

✗ DON'T

  • #409EFF를 현대 브랜드 색으로 사용 — Element UI 기본값
  • Head/Text 폰트를 같은 패밀리로 통일
  • 현대 사이트에서 라운드 버튼 남발
