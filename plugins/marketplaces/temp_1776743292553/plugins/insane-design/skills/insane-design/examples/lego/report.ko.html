LEGO — 실제 CSS 기반 분석

lego.com의 디자인 시스템 분석 — 브랜드 레드(#E3000B) CTA, 순백 배경, Cera Pro 폰트의 현대적이고 활기찬 장난감 브랜드 미학.

Quick Start 5분 안에 LEGO처럼 만들기
1 · 폰트 body { font-family: "Cera Pro", "Nunito", "Helvetica Neue", Arial, sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #FFFFFF; --fg: #1A1A1A; } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 레드 :root { --brand: #E3000B; --brand-yellow: #FFD700; } h1 { font-weight: 900; text-transform: uppercase; }
절대 하지 말아야 할 것 하나: 레드(#E3000B)를 위험/에러 색으로만 취급하는 것. LEGO 레드는 primary CTA 버튼이자 브랜드 identity다. 에러로만 쓰면 LEGO의 긍정적 브랜드 에너지가 사라진다.

13 · Do / Don't

Do / Don't

LEGO를 LEGO답게 만드는 규칙.

DO

  • CTA는 레드(#E3000B) — LEGO 브랜드의 핵심 CTA 색상, 긍정적 역할
  • 헤드라인은 weight 900 + uppercase — 굵고 대문자 스타일이 LEGO 정체성
  • 노랑(#FFD700)은 연령 배지·보조 포인트 전용
  • 제품 카드에 연령/조각 수 배지 필수 — LEGO 필수 정보
  • 배경은 순백(#FFFFFF) — 제품 이미지 대비 최대화

DON'T

  • 레드(#E3000B)를 에러 색으로만 쓰지 않는다 — LEGO에서 레드는 긍정적 CTA
  • 노랑(#FFD700)을 배경 전면 사용하지 않는다 — 포인트 컬러 역할
  • border-radius 24px 이상 과도 사용 금지 — 너무 둥글면 브랜드 tone과 불일치
  • 폰트를 Comic Sans 계열로 대체하지 않는다 — Cera Pro는 정밀·현대적
  • 다크 배경(#313131)을 기본으로 쓰지 않는다 — LEGO는 밝고 활기찬 라이트 우선
