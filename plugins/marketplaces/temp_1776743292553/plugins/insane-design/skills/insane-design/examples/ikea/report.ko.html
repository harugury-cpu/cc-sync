IKEA — 실제 CSS 기반 분석

ikea.com의 디자인 시스템 분석 — IKEA 옐로우(#FFDB00) 브랜드 색상, 검정 CTA 버튼, Noto IKEA 전용 폰트, 스웨덴 블루-옐로우 정체성.

Quick Start 5분 안에 IKEA처럼 만들기
1 · 폰트 body { font-family: "Noto IKEA", "Noto Sans", Roboto, "Open Sans", sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #FFFEFB; --fg: #111111; } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 옐로우 :root { --brand: #FFDB00; --ikea-blue: #0058A3; } /* Primary btn = 검정 */ .btn { background: #111111; }
절대 하지 말아야 할 것 하나: 파란색(#0058A3)을 primary CTA로 쓰는 것. IKEA 파란색은 보조 브랜드 색상이고 primary 버튼은 검정(#111111)이다. 옐로우(#FFDB00)는 배지·특별 CTA용으로만 사용한다.

13 · Do / Don't

Do / Don't

IKEA를 IKEA답게 만드는 규칙.

DO

  • Primary 버튼은 검정(#111111) — CSS selector_role 직접 추출값
  • 옐로우(#FFDB00)는 배지·특별 프로모션 CTA — 브랜드 식별자 역할
  • 배경은 near-white #FFFEFB — 순백이 아닌 약간 따뜻한 크림
  • 파란색(#0058A3)은 링크·정보 역할 — 스웨덴 국기 color 보조 사용
  • 제품명은 스웨덴어 또는 조어 대문자 (BILLY, KALLAX)

DON'T

  • 파란색(#0058A3)을 primary CTA로 쓰지 않는다 — 버튼은 검정
  • 옐로우(#FFDB00)를 일반 텍스트 배경으로 깔지 않는다 — 배지 전용
  • Svelte 해시 클래스(.svelte-*)를 직접 참조하지 않는다 — 빌드 시 변경
  • 그림자를 과도하게 쓰지 않는다 — IKEA는 flat 디자인 위주
  • weight 900+ 헤드라인을 일반 텍스트에 쓰지 않는다 — 700이 최대
