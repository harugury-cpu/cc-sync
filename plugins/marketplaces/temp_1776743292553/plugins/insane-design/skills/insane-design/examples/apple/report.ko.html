Design System Report

Apple — SkinKit (--sk-*)

https://apple.com 이 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. 모든 값은 CSS 커스텀 프로퍼티를 파싱한 결과이고, AI 추론이나 추측은 섞이지 않았다.

Quick Start 5분 안에 Apple처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight body { font-family: "SF Pro Text", "SF Pro Icons", -apple-system, BlinkMacSystemFont, "Helvetica Neue", sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #F5F5F7; /* 오프화이트 */ --fg: #1D1D1F; /* 따뜻한 다크 */ } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 블루 :root { --brand: #0071E3; /* CTA · focus ring 모두 이 색 */ }
절대 하지 말아야 할 것 하나: 배경을 #FFFFFF(순백)으로 두는 것. Apple의 실제 페이지 배경은 #F5F5F7이다. 순백으로 두면 Apple 특유의 고급스러운 공기감이 사라지고 그냥 평범한 흰 페이지처럼 보인다.

03 · Typography

타이포그래피

SF Pro Text(본문)와 SF Pro Display(헤딩)를 구분하는 것이 Apple 타이포의 핵심이다. Heading에는 반드시 negative letter-spacing을 적용해야 한다.

Font stack
Display font SF Pro Display — Apple 전용, 재배포 불가. 헤딩에 사용.
Body font SF Pro Text — Apple 전용, 본문에 사용.
Fallback -apple-system, BlinkMacSystemFont, "Helvetica Neue", Helvetica, Arial
Weight normal / bold 400 / 600
라이선스 주의: SF Pro는 Apple 기기에만 번들됨. 재배포 불가. 비Apple 환경에서는 -apple-system, BlinkMacSystemFont fallback이 동일 폰트를 로드한다. 무료 대체재로는 Inter가 근접하다.

05 · Spacing

스페이싱

SkinKit의 focus ring offset이 핵심 spacing 토큰이다. Apple은 4px 배수 그리드를 기반으로 하되, 섹션 간격은 넉넉히 52px을 쓴다.

SkinKit focus token: --sk-focus-offset: 1px (전체 focus ring 오프셋), --sk-button-focus-offset: 3px (버튼 컨테이너 focus offset). 접근성 원칙상 이 값을 0으로 두지 않는다.

08 · Writing Principles

라이팅 원칙

Apple의 카피는 간결하고 감성적이다. 기술 용어는 최소화하고, 마침표는 생략한다. CTA는 직접적인 동사형으로 써야 한다.

Headline
간결하고 감성적. 마침표 없음. 제품 혜택이 아닌 경험을 말한다.
당신의 삶을 바꿀 iPhone.
Primary CTA
직접적 동사형. 즉각적 행동을 유도한다.

정보 탐색 유도. 화살표(>)를 붙여 링크임을 명시한다.

Tone
친근하지만 프리미엄. 기술 용어는 최소화. 사용자를 존중하는 어조.

10 · Do / Don't

Do / Don't

Apple 디자인 시스템을 구현할 때 가장 자주 틀리는 항목들.

DO

  • 페이지 배경은 #F5F5F7을 써라 — 순백이 아닌 Apple의 시그니처 오프화이트다.
  • 텍스트는 #1D1D1F를 써라 — 순흑이 아닌 따뜻한 다크다.
  • CTA 버튼에 3단계 상태를 모두 구현하라: #0071E3 → #0076DF → #006EDB.
  • focus ring 색도 #0071E3으로 CTA와 일치시켜라 — SkinKit 원칙이다.
  • 큰 heading에 letter-spacing: -0.02em을 적용하라.
  • CTA 버튼은 pill shape(border-radius 980px)으로 만들어라.

DON'T

  • 배경을 #FFFFFF로 두지 마라 — Apple의 배경은 #F5F5F7이다.
  • CTA 버튼을 정사각형으로 만들지 마라 — Apple은 pill shape(980px)이다.
  • SF Pro를 재배포하지 마라 — 라이선스 위반이다. -apple-system, BlinkMacSystemFont를 쓰라.
  • 다크 배경 링크에 #0071E3을 그대로 쓰지 마라 — 다크에서는 #2997FF로 교체해야 한다.
  • heading에 font-weight: 400을 쓰지 마라 — Apple heading은 600이다.
Common Mistakes
틀린 값 올바른 값 이유
#FFFFFF #F5F5F7 Apple 페이지 배경은 오프화이트. 순백은 의도된 배경이 아니다.
border-radius: 8px border-radius: 980px CTA 버튼은 pill shape이다. 8px은 Apple 카드용이다.
#0071E3 (dark bg link) #2997FF 다크 배경에서는 대비를 위해 밝은 blue로 교체해야 한다.
font-weight: 400 (heading) font-weight: 600 Apple heading은 600이다. 400은 본문 전용이다.
letter-spacing: 0 (heading) -0.01em ~ -0.02em Apple heading에는 반드시 negative letter-spacing이 들어간다.
