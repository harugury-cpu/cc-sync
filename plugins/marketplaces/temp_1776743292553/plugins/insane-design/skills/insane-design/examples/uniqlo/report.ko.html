Design System Report

UNIQLO — system-ui · 흑백 UI + 로고 Red

https://www.uniqlo.com 이 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. #FF0000은 로고와 세일 배지에만 — UI 동작(버튼·링크)은 순수한 흑백이다.

Quick Start 5분 안에 UNIQLO처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight 400 body { font-family: system-ui, -apple-system, "Helvetica Neue", Arial, sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #FFFFFF; /* 순백 */ --fg: #000000; /* 순흑 */ } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 컬러 (로고 전용) :root { /* 로고·sale 배지에만 사용 */ --brand: #FF0000; /* CTA 버튼은 반드시 검정 */ --btn-bg: #000000; }
절대 하지 말아야 할 것 하나: #FF0000을 CTA 버튼이나 본문 링크 색상으로 쓰는 것. UNIQLO의 빨간색은 로고 SVG와 할인 배지에만 존재한다. UI의 모든 동작(버튼·링크·호버)은 흑백으로만 처리된다.

03 · Typography

타이포그래피

폰트 스택, 굵기 정의, 타이포그래피 스케일.

Font stack
Display font system-ui (OS 기본 — 외부 폰트 없음)
Fallback chain -apple-system, BlinkMacSystemFont, "Helvetica Neue", Arial, sans-serif
Weight normal / bold 400 / 700
다국어 시스템 폰트 전략: UNIQLO는 일본어(Hiragino, Yu Gothic), 한국어(Apple SD Gothic Neo), 영어(Helvetica Neue)를 모두 OS 기본 폰트로 처리한다. 외부 폰트 로딩이 없으므로 로드 타임이 빠르고, 각 지역 OS의 최적 폰트가 자동 적용된다.

08 · Layout Patterns

레이아웃 패턴

Hero, 상품 그리드, 섹션 리듬. 4열 그리드가 UNIQLO의 시그니처.

Hero

  • 전체폭 배경, 상품 이미지 중심 구성
  • 배경: #FFFFFF — 제품이 주인공
  • H1: 3rem / weight 700 / tracking -0.02em
  • max-width: 1440px

Product Grid

  • Desktop: 4열 그리드 (UNIQLO 시그니처)
  • Tablet(1024px): 2열 그리드
  • Mobile(768px): 1열 리스트
  • 카드 border-radius: 0px 직각

Section Rhythm

section {
  padding: 64px 40px;
  max-width: 1440px;
}

Breakpoints

  • Mobile: 768px — 1열
  • Tablet: 1024px — 2열
  • Desktop: 1440px — 4열

12 · Do / Don't

Do / Don't

UNIQLO 디자인 시스템의 핵심 규칙. 이것만 지켜도 80% 이상 일치.

✓ DO

  • primary CTA는 background: #000000으로 유지한다
  • #FF0000은 로고·sale 배지·프로모션 배너에만 사용한다
  • heading은 모두 font-weight: 700을 적용한다
  • border-radius: 0으로 직각 형태를 유지한다
  • 상품 그리드는 4열(desktop) 구성을 기본으로 한다
  • 시스템 폰트를 그대로 사용한다 — 외부 폰트 금지

✗ DON'T

  • #FF0000을 CTA 버튼이나 링크 색상으로 쓰지 않는다
  • 중간 weight(500, 600)를 heading에 쓰지 않는다 — 400 또는 700만
  • border-radius에 큰 값(8px 이상)을 넣지 않는다
  • 배경을 회색(#F5F5F5)으로 두지 않는다 — 기본 배경은 순백
  • 불필요한 색상 악센트를 UI에 추가하지 않는다 — 흑백 시스템
잘못된 구현
올바른 구현
이유
btn { background: #FF0000 }
btn { background: #000000 }

border-radius: 8px
border-radius: 0px

font-weight: 600
font-weight: 400 or 700
중간 weight는 거의 사용하지 않는다. 400/700 두 가지만.
background: #F5F5F5
background: #FFFFFF
UNIQLO 배경은 순백. 회색 배경은 다른 브랜드 느낌.
