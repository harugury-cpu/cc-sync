Design System Report

Chanel — --color-*

https://www.chanel.com 이 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. 순흑(#000000) CTA와 near-white 배경(#F9F9F9)의 럭셔리 모노크롬 시스템.

Quick Start 5분 안에 Chanel처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight body { font-family: "abchanel-2022", "helvetica-adjusted-abchanel-2022", helvetica, sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #F9F9F9; /* near-white */ --fg: #000000; } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 컬러 :root { --color-primary: #000000; --color-background-variant: #F9F9F9; /* CTA text = #F9F9F9 */ }
절대 하지 말아야 할 것 하나: primary 버튼 배경을 #767676으로 두는 것. --color-secondary-light: #767676은 hover/active 상태 전용이다. 기본 CTA는 반드시 background: #000000.

    11 · Do / Don't

    Do / Don't

    지켜야 할 것과 피해야 할 것.

    Do

    • primary button은 background: #000000, color: #F9F9F9로 설정한다
    • hover 상태는 background: #767676으로만 처리한다
    • border-radius: 0을 모든 버튼·카드에 유지한다
    • body weight는 400, heading 디스플레이는 300을 쓴다
    • 배경은 #F9F9F9(살짝 따뜻한 near-white)로 설정한다
    • CSS 변수 --color-primary, --color-background-variant를 사용한다

    Don't

    • background: #767676을 기본 버튼에 쓰지 않는다 — hover 전용
    • border-radius에 큰 값을 넣지 않는다
    • 채도 높은 악센트 색상을 UI에 추가하지 않는다
    • font-weight: 700을 body나 heading에 쓰지 않는다
    • 배경을 순백(#FFFFFF)으로 두지 않는다 — #F9F9F9 사용
