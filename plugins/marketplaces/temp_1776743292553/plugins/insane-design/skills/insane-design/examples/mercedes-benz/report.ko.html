Mercedes-Benz — body weight 100

mercedes-benz.com의 가장 큰 비밀: font-weight가 100(Ultra Light)이다. CSS에서 23회 감지. 이 초경량 타이포그래피가 럭셔리의 핵심.

Quick Start 5분 안에 Mercedes-Benz처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight 100 body { font-family: "MBCorpoSText", "MBCorpoAText", "Helvetica Neue", Arial, sans-serif; font-weight: 100; /* ⚠️ 400 아님. MB는 100 */ }
2 · 배경 + 텍스트 :root { --bg: #FFFFFF; --fg: #000000; } body { background: var(--bg); color: var(--fg); }
3 · 실버 미드 포인트 :root { /* 실버 — 링크/보조 텍스트 */ --muted: #9F9F9F; /* 모노크롬 팔레트 */ }
절대 하지 말아야 할 것 하나: body를 font-weight: 400으로 두는 것. Mercedes-Benz의 정체성은 weight 100 (Ultra Light)이다. CSS에서 23회 등장하며, 이 초경량이 럭셔리 브랜드를 만든다.

03 · 컬러

실버 미드 — 모노크롬의 핵심

CSS에서 감지된 컬러: #9F9F9F(12회), #000000(2회). 완전한 모노크롬 팔레트.

CSS 감지 팔레트
실버 미드 (12회)#9F9F9F
순흑 (2회)#000000
#9F9F9F은 네비게이션 링크 컬러(brandhub-search-link-list-item)에서 가장 많이 사용. 이 실버 미드 톤이 Mercedes의 금속 럭셔리감을 UI에 반영한다. transition: color .4s ease-in-out으로 부드럽게 전환.

05 · CSS 스니펫

Drop-in CSS

/* Mercedes-Benz — copy into your root stylesheet */
:root {
  /* Fonts */
  --mb-font-family: "MBCorpoSText", "MBCorpoAText",
    "Helvetica Neue", Arial, sans-serif;
  --mb-font-weight-normal: 100;   /* ⚠️ Ultra Light */
  --mb-font-weight-bold: 400;

  /* Brand (monochrome) */
  --mb-color-silver: #9F9F9F;
  --mb-color-black: #000000;
  --mb-color-white: #FFFFFF;

  /* Surfaces */
  --mb-bg-page: #FFFFFF;
  --mb-bg-dark: #000000;
  --mb-text: #000000;
  --mb-text-muted: #9F9F9F;

  /* Spacing observed */
  --mb-space-nav: 1.25rem;
  --mb-font-size-nav: 1.1428571429rem;
  --mb-nav-transition: color .4s ease-in-out;
}
06 · Do / Don't

자주 틀리는 것들

✓ DO

  • body에 font-weight: 100 사용 — 이것이 Mercedes의 핵심
  • 네비게이션 링크 색으로 #9F9F9F 사용
  • 링크 hover에 transition: color .4s ease-in-out 적용
  • 레이아웃 최대한 여백 넓게, 미니멀하게

✗ DON'T

  • body를 font-weight: 400 이상으로 쓰지 말 것
  • 컬러풀한 accent 추가하지 말 것 — 모노크롬이 정체성
  • 버튼 모서리를 둥글게 만들지 말 것
  • #9F9F9F를 CTA 버튼에 쓰지 말 것 — 보조 텍스트 색
