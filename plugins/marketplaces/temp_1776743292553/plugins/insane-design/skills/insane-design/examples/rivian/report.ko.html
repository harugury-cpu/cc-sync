Design System Report

Rivian — earth tone EV

rivian.com의 정체성은 어스 톤(earth tone)이다. #E6E5DF(따뜻한 오프화이트)와 #465243(포레스트 그린)이 테슬라식 순흑/순백과 완전히 다른 feel을 만든다. Tailwind v4 기반 semantic 토큰 시스템 구축.

Quick Start 5분 안에 Rivian처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight body { font-family: Adventure, "HelveticaNeue", "Helvetica Neue", Helvetica, sans-serif; font-weight: 400; }
2 · Earth tone 배경 :root { --bg: #FFFFFF; /* earth 섹션 */ --bg-earth: #E6E5DF; --fg: #000000; } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 그린 :root { /* Forest Green */ --brand: #465243; /* Amber accent */ --amber: #FFAC00; /* Near-black */ --near-black: #151515; }
절대 하지 말아야 할 것: Rivian을 테슬라처럼 순흑/순백 모노크롬으로 만드는 것. Rivian의 정체성은 어스 톤이다. #E6E5DF(따뜻한 오프화이트)와 #465243(포레스트 그린)이 실제 feel을 만든다.

핵심 포인트: Rivian은 전기차 브랜드임에도 테슬라의 차가운 순흑/순백 미학 대신 따뜻한 어스 톤을 채택했다. #E6E5DF(11회)는 warm off-white로, 순흑 #F2F2F2보다 더 따뜻하다. 이 차이가 어드벤처 브랜드 정체성을 만든다.

CSS 스니펫

루트 스타일시트에 복사해서 바로 쓸 수 있는 Rivian 디자인 토큰.

/* Rivian — copy into your root stylesheet */
:root {
  /* Fonts */
  --rivian-font-family: Adventure, "HelveticaNeue",
    "Helvetica Neue", Helvetica, Arial, sans-serif;
  --rivian-font-family-mono: "Adventure Mono",
    ui-monospace, monospace;
  --rivian-font-weight-normal: 400;
  --rivian-font-weight-bold: 600;

  /* Brand (earth tone) */
  --rivian-color-brand-green: #465243;
  --rivian-color-earth-white: #E6E5DF;
  --rivian-color-amber: #FFAC00;

  /* Surfaces */
  --rivian-bg-page: #FFFFFF;
  --rivian-bg-earth: #E6E5DF;
  --rivian-bg-dark: #000000;
  --rivian-bg-near-black: #151515;
  --rivian-text: #000000;
  --rivian-text-muted: #494949;
}

/* Semantic layer (Tailwind v4 style) */
:root {
  --surface-primary: #FFFFFF;
  --surface-primaryAlt: #FAFAFA;
  --text-primary: #000000;
  --stroke-primary: #000000;
}
[data-theme="dark"] {
  --surface-primary: #000000;
  --surface-primaryAlt: #151515;
  --text-primary: #FFFFFF;
  --stroke-primary: #FFFFFF;
}

06

Do / Don't

Rivian 디자인을 구현할 때 지켜야 할 핵심 규칙.

✓ DO

  • 어스 톤 #E6E5DF를 배경 섹션에 사용
  • 브랜드 그린 #465243을 절제적인 accent로
  • light/dark 테마를 --surface-primary, --text-primary로 교체
  • 폰트는 Adventure — 없으면 DM Sans 또는 Manrope
  • font-weight bold는 600 (700 아님)

✗ DON'T

  • 순흑/순백 모노크롬만으로 레이아웃 구성하지 말 것
  • Amber #FFAC00을 primary CTA에 쓰지 말 것
  • 브랜드 그린을 배경 전체에 깔지 말 것
  • 텍스트에 순흑 #000000 대신 #151515 우선 사용
