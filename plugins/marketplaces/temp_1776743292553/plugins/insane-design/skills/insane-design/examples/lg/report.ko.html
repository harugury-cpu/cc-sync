LG — LGSmHaTSB · 레드 계열 4종

www.lg.com/kr/ 의 CSS. LGSmHaTSB 단일 폰트, weight 700 기본. 레드 계열 #B31045 / #C00000 / #DD3C70 / #EBB3A4 — 4종 공존.

Quick Start5분 안에 LG처럼 만들기
1 · 폰트 + weight 700body { font-family: "LGSmHaTSB", sans-serif; font-weight: 700; /* ⚠️ 400 아님 */ }
2 · 배경 + 텍스트:root { --bg: #FFFFFF; --fg: #111111; } body { background:var(--bg); color:var(--fg); }
3 · 브랜드 레드:root { --brand: #B31045; --btn-go: #DD3C70; --btn-down: #EBB3A4; }
절대 하지 말아야 할 것 하나: 단일 레드 색 #FF0000 사용. LG CSS에는 순수 빨강이 없다. #B31045(brand) / #DD3C70(btn.go) / #EBB3A4(btn.down) — 3가지 레드 계열이 역할별로 분리된다.

08 · Do / Don't

자주 틀리는 것들

LG 웹 구현 시 가장 많이 실수하는 패턴.

✓ DO

  • LG 폰트 weight 700 기본값
  • 레드 구분: brand(#B31045), btn-go(#DD3C70), btn-down(#EBB3A4)
  • letter-spacing -0.02em (버튼)
  • 본문 텍스트 #555555

✗ DON'T

  • #FF0000 순수 빨강 — LG CSS에 없음
  • 레드 4종을 하나로 통일 — 역할이 다름
  • 버튼에 border-radius 추가 — 각진 직사각형이 기본
  • weight 400 — LG는 700이 기본
